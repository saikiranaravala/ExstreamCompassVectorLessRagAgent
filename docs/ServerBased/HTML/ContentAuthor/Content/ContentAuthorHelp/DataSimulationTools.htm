Using the Content Author data simulation tools

If your communication contains variables, Content Author allows you to temporarily change the values of those variables to simulate how the communication behaves in different data scenarios. By default, Content Author uses a static set of sample data when you add variables to your content—meaning that each variable has a sample value associated with it. As you incorporate business rules to include or exclude content based on changing scenarios, however, you will likely have difficulty using static data to test how your rules and variables behave in different data conditions. For example, suppose that you have a communication that contains a graphic message that is targeted for customers who live in the state of Florida. If the sample data in your communication uses Georgia as the default state value, then the graphic message will not appear when you preview the communication in Communications view.

Depending on the design of your communication, you might also be able to switch sample data sets, which allows you to see how your communication looks with a potentially completely different set of variables. This functionality is especially useful when the available frames or number of pages in a communication might change when the data set changes, such as when regulations in a particular state require additional content to be included in a communication.

Important: In communications that contain variables, Simulation view is always available. The availability of sample data sets, on the other hand, is completely dependent on the design of your communication. If you have questions about the communication you are working on, you should talk to the person who designed it.

Change variable values in Simulation view

  1. In Communications view, click Simulation just above the Selected Communication card.

  2. In the Simulation view, in the Data panel, change the sample variable values as needed, and then click Run.

    The variable values are updated in the communication in Simulation view, and updated variable values are shaded green in the Data panel.

  3. (Optional) To reset the variables to their original sample values, click Reset.

  4. (Optional) Depending on how your communication was designed, you can switch between print and email previews of the communication by clicking PDF and HTML, respectively.

  5. (Optional) Click to download a copy of the currently selected simulation.

  6. When you are finished with the simulation, click Exit simulation, just above the Data panel.

    Simulation view closes and you are returned to Communications view. You can also use the hamburger menu () to return directly to the Content Launcher.

Change sample data sets

  1. In Communications view, click on the toolbar.

  2. Select a sample data file from the list. The current sample data file is indicated by a blue check mark.

  3. Click Select.

    The communication changes to match the sample data file that you selected.
