Getting started in Content Author

Exstream Content Author is a web application that allows you to create or edit individual pieces of content that are used as part of larger customer communications. Much like a standard word processing program, Content Author lets you enter or edit text, change basic formatting, and add tables and images to messages and clauses. The controls and icons are probably mostly familiar to you—for those that are not, simply point to a button or control to see a brief description.

Content Author is particularly powerful in its ability to let you use variables and rules to make sophisticated content that targets individual customers. Using variables—and rules based on variables—lets you ensure that your customers get only the information that is pertinent to them. Best of all, you do not need intervention from IT to update customer communications.

At a high level, the basic workflow for Content Author is as follows: A designer in your company puts together a communication that will eventually be sent to customers. Within that communication, the designer includes one or more editable frames. And within those frames, you add messages or clauses in Content Author. When you finish making updates to the content, you save your changes in Content Author and then your new content makes its way into the communication that is sent to customers, or it is included in content that is editable in Empower Editor. Depending on the design, the final customer communication might include only the messages or clauses that you worked on, or it might include other elements that are added dynamically based on customer data.

Content Author provides the tools that you need to easily create or update content. If you have questions about those tools, you can find answers here in this help. However, if you ever have questions about a design itself—from why something looks the way it does, to why you cannot make changes to certain content—you should talk to the person who designed the communication.

Localized user interface and online help

The Content Author user interface and online help are available in the following languages:

  • Brazilian Portuguese
  • Chinese
  • Dutch
  • English
  • French
  • German
  • Japanese
  • Spanish

If your web browser is configured to use one of the available languages, Content Author derives its language settings from your web browser.

Keyboard shortcuts

You can use the following keyboard shortcuts in Content Author as an alternative way to perform some common tasks that are typically performed using a mouse or another input device.

General shortcuts
Keyboard shortcut Action
ALT+F1 Open Content Author Help.
CTRL+C Copy the selected content.
CTRL+X

Cut the selected content.
CTRL+V Paste copied content from the clipboard.
CTRL+Z Undo the previous action.
CTRL+Y Redo the previous action.
TAB

Pressing this keyboard shortcut will do the following things, depending on where you are in Content Author at the time:

  • Increase the indent of selected text.
  • Navigate to the next cell in a table.
  • In selection mode, select the next object in the frame.
  • When typing text, insert a tab stop.
SHIFT+TAB

Pressing this keyboard shortcut will do the following things, depending on where you are in Content Author at the time:

  • Decrease the indent of selected text.
  • Navigate to the previous cell in a table.
  • In selection mode, select the previous object in the frame.
Formatting shortcuts
Keyboard shortcut Action
CTRL+B Make the text bold.
CTRL+I Make the text italic.
CTRL+U Underline the text.
CTRL+[ Increase the indent for a paragraph.
CTRL+] Decrease the indent for a paragraph.
CTRL+. Make the text superscript.
CTRL+, Make the text subscript.
Navigation shortcuts
Keyboard shortcut Action
CTRL+<left arrow> In edit mode, move the cursor one word to the left.
CTRL+<right arrow> In edit mode, move the cursor one word to the right.
CTRL+<up arrow> In edit mode, move the cursor to the beginning of the paragraph.
CTRL+<down arrow> In edit mode, move the cursor to the end of the paragraph.
Positioning shortcuts
Keyboard shortcut Action
<arrow keys> In selection mode, move the object 1 pixel to the right, left, up, or down.
SHIFT+<arrow keys> In selection mode, move the object 10 pixels to the right, left, up, or down.
Resizing shortcuts
Keyboard shortcut Action
CTRL+<right arrow> With an object (such as an image or text box) selected, increase the width of the object by 1 pixel. If the Preserve aspect ratio check box is selected, the width and height will increase proportionally.
CTRL+SHIFT+<right arrow> With an object (such as an image or text box) selected, increase the width of the object by 10 pixels. If the Preserve aspect ratio check box is selected, the width and height will increase proportionally.
CTRL+<left arrow> With an object (such as an image or text box) selected, decrease the width of the object by 1 pixel. If the Preserve aspect ratio check box is selected, the width and height will decrease proportionally.
CTRL+SHIFT+<left arrow> With an object (such as an image or text box) selected, decrease the width of the object by 10 pixels. If the Preserve aspect ratio check box is selected, the width and height will decrease proportionally.
CTRL+<down arrow> With an object (such as an image or text box) selected, increase the height of the object by 1 pixel. If the Preserve aspect ratio check box is selected, the width and height will increase proportionally.
CTRL+SHIFT+<down arrow> With an object (such as an image or text box) selected, increase the height of the object by 10 pixels. If the Preserve aspect ratio check box is selected, the width and height will increase proportionally.
CTRL+<up arrow> With an object (such as an image or text box) selected, decrease the height of the object by 1 pixel. If the Preserve aspect ratio check box is selected, the width and height will decrease proportionally.
CTRL+SHIFT+<up arrow> With an object (such as an image or text box) selected, decrease the height of the object by 10 pixels. If the Preserve aspect ratio check box is selected, the width and height will decrease proportionally.
Selection shortcuts
Keyboard shortcut Action
ESC In edit mode, changes to selection mode and selects the current object. In selection mode, deselects the currently selected object.
CTRL+A In edit mode, selects all text and embedded objects. In selection mode, selects all available objects.
SHIFT+<right arrow> In edit mode, select the character to the right of the cursor.
SHIFT+<left arrow> In edit mode, select the character to the left of the cursor.
SHIFT+<down arrow> In edit mode, select the next line of text up to the current cursor position.
SHIFT+<up arrow> In edit mode, select the previous line of text up to the current cursor position.
CTRL+SHIFT+<right arrow> In edit mode, from the current cursor position, select all remaining characters in a word, including any spaces or tab stops that follow.
CTRL+SHIFT+<left arrow> In edit mode, from the current cursor position, select the previous characters in a word.
CTRL+SHIFT+<down arrow> In edit mode, from the current cursor position, select the remaining lines of text in a paragraph, including any spaces or tab stops that follow.
CTRL+SHIFT+<up arrow> In edit mode, from the current cursor position, select the previous lines of text in a paragraph, including any spaces or tab stops that precede it.
