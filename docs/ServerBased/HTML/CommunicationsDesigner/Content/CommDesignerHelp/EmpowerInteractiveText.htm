Adding interactive text for Empower users

In Design view, you can set interactive properties on text boxes, as well as on embedded text boxes and on the inline text content itself in text boxes. You can also allow Empower users to change variable values or to make selections based on variables. Keep in mind that you cannot apply interactive properties both to the text box and to any static content that is inside it at the same time. You can set interactive properties on either text boxes, or on the inline text content inside a text box.

Note: Interactive Empower properties are supported only in page and email designs. The option to set Empower properties is not available for web designs.

Setting Empower properties for text boxes

  1. In the design area, select the text box that you want to make editable in Empower Editor.

  2. In the object properties panel, click the Empower properties button .

    Note: If you set interactive properties for a parent design object, you cannot set different interactive properties for individual static text selections within that design object.

  3. In the Empower card, in the Changes allowed list, select one of the following options to specify whether users can edit the selection in Empower Editor:

    • No changes—Users cannot edit this selection in Empower Editor. This is the default option.

    • Change is optional—Users can decide whether to change the contents of the selection.

    • Change is required—Users must make changes to the selection in Empower Editor before it is included in customer output. You might select this option if, for example, a selected text box contains placeholder text that must be replaced with customer-specific content before output is generated.

      Important: If you embed tables or images in an interactive design object, those embedded objects will not be editable in Empower Editor; however, Empower users will be able to delete those items.

  4. If you enable Empower users to make changes to the selection, specify the following interactive properties as needed:

    • To allow changes to both the text content and the text formatting, select the Allow changes to text formatting check box.

    • Optionally, to include an informational tooltip that will appear when users point to the selected content in Empower Editor, enter a tooltip title in the Tooltip title box, and enter the body of the tooltip in the Tooltip description box.

Setting Empower properties and selection controls for inline text

To streamline the editing experience for users in Empower Editor, and to provide users with convenient options for entering data, you can add selection controls to inline interactive text within text boxes. Selection controls—such as drop-down lists, calendars, and free-form entry boxes—let users select from multiple options, instead of manually entering a text string.

To set Empower properties and add a selection control to inline text:

  1. In the design area, select the inline text.

  2. On the toolbar, click the Empower edit area button .

    Note: You can set interactive properties for multiple text selections within a design object, but not for overlapping selections, or for selections that span multiple paragraphs.

  3. In the Changes allowed list, select one of the following options to specify whether users can edit the selection in Empower Editor:

    • No changes—Users cannot edit this selection in Empower Editor. This is the default option.

    • Change is optional—Users can decide whether to change the contents of the selection.

    • Change is required—Users must make changes to the selection in Empower Editor before it is included in customer output.

  4. In the Add selection control list, select one of the following options:

    • None—This is the default option. Users must manually enter text in Empower Editor.

    • Use drop-down list—Users can select the content that they want to enter from a list. You must create this selection list in Communications Designer. To create the selection list:

      1. To open the Interactive list dialog box, click Create list.

      2. In the Add new item area, enter list items as needed, pressing ENTER after each entry, then click Save.

        The list items are associated with the selected interactive text and will appear in the Empower document when you create output. To edit or delete the list, click Edit list in the Apply Empower settings dialog box.

    • Calendar—Users must select a date from a graphical representation of a calendar.

    • Calendar and free-form entry—Users can either select a date from a graphical representation of a calendar, or they can type the date in a text box.

  5. If you enable Empower users to make changes to the selection, specify the following interactive properties as needed:

    • To allow changes to both the text content and the text formatting, select the Allow changes to text formatting check box.

    • Optionally, to include an informational tooltip that will appear when users point to the selected content in Empower Editor, enter a tooltip title in the Tooltip title box, and enter the body of the tooltip in the Tooltip description box.

Setting Empower properties and selection controls for variables

To provide an easy way for Empower users to change variable values or make selections based on variables, you can specify a list of options from which users can select text, number, and currency variables. You can also select the data entry type for date variables. For Boolean variables, you can update only the formatting and tooltip options, and in Empower Editor, users can select only a true or false value for the variable. Keep in mind that for array variables, you can update only the formatting options and specify whether changes to the content are required or optional. In addition, for array variables, users can change only one value at a time in Empower Editor.

To set Empower properties and add a selection control to variables:

  1. Double-click the variable in the content to select it, and then in the Variables panel, click the Empower properties button .

    Note: Unlike inline text selections, you can set interactive properties for inline variables even if those variables are within design objects that have interactive properties.

  2. In the Changes allowed list, select one of the following options to specify whether users can edit the selection in Empower Editor:

    • No changes—Users cannot edit this selection in Empower Editor. This is the default option.

    • Change is optional—Users can decide whether to change the contents of the selection.

    • Change is required—Users must make changes to the selection in Empower Editor before it is included in customer output.

  3. Specify a list of options or a data entry type for the variable by completing one of the actions described in the following table:

    Action Description
    Specify a list of options for text, number, or currency variables.

    1. Select the Use drop-down list check box, and then click Create list to open the Interactive list dialog box.

    2. In the Add new item area, enter list items as needed, pressing ENTER after each entry, and then click Save.

      The list items are associated with the variable and will appear in the Empower document when you create output. To edit or delete the list, click Edit list.
    Specify a data entry type for date variables.

    In the Data-entry type list, select one of the following options:

    • Calendar—Users can select a date from a graphical representation of a calendar.

    • Calendar and free-form entry—Users can either select a date from a graphical representation of a calendar, or they can type the date in a text box.

    • Free-form entry only—Users can type the date in a text box.

  4. If you enable Empower users to make changes to the selection, specify the following interactive properties as needed:

    • To allow changes to both the text content and the text formatting, select the Allow changes to text formatting check box.

    • Optionally, to include an informational tooltip that will appear when users point to the selected content in Empower Editor, enter a tooltip title in the Tooltip title box, and enter the body of the tooltip in the Tooltip description box.
