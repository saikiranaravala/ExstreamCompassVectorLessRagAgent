Starting a new communication

To open Structure view in Communications Designer and start designing your communications, use the Communications page in Content Launcher to create a new communication or select an existing communication for editing.

To access the Communications page, in Content Launcher, on the toolbar, click the Communications button .

Prerequisites for creating new communications

Make sure that you meet the following prerequisites for creating a communication:

  • Design Manager application with design pack—Before you can create a new communication in Communications Designer, you must have first created and uploaded a Design Manager application that includes an appropriate design pack.
  • Communications Designer data source—(Optional) If you added a Communications Designer data file to your Design Manager application, you must also create a data source for use with the new communication in Communications Designer. You must create this data source before you can create a communication that will use the same design pack as the data source.

Important: After you create a communication, you cannot associate it with a different design pack or data source. Make sure that you are selecting the correct design pack when you create a new communication. If you plan to use a Communications Designer data source, you must also make sure that you have previously created and associated the data source with the same design pack. You can update the associated design pack and data source after the communication has been created.

For information about setting up your application correctly in Design Manager, and uploading the packaged application to make the required resources available in Communications Designer, see Designing for Exstream Web Applications in OpenText Exstream Design and Production Online Help.

Creating a new communication

  1. In the Communications Designer widget, on the Communications tab, click the New button to see a list of available design packs.

  2. In the Design pack name column, select a design pack, and click Next.

  3. If prompted, in the Data source name column, select a data source, and click Next.

  4. Specify a name, and optionally, a description for the communication that you want to create.

  5. Click Create.

    The new communication opens in Structure view, and the name of the communication appears in the Structure map.

Opening an existing communication

  1. In the Communications Designer widget, click one of the following buttons:

    • Existing—Displays all of the existing communications

    • Recent—Displays a list of communications that you have recently opened in Communications Designer

    • Favorites—Displays a list of communications that have been bookmarked

      Tip: If you work frequently with a specific communication, you can bookmark that communication so that it appears in the Favorites list. To bookmark a communication, point to the communication in the Existing or Recent tabs, and click the Bookmark icon in the Favorite column.

  2. Select the communication that you want to open, and click Open.

    The communication opens in Structure view, and the name of the communication appears in the Structure map.
