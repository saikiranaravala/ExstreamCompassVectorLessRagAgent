Updating component properties

In Structure view, you can see all of the existing reusable design objects—called components—that are used in each design. Components are an easy way to reuse design objects across multiple designs.

In addition to the properties that are described in this section, you can also use Structure view to view the workflow history for each component.

Updating basic properties

In the Properties panel, you can use the General tab to rename a component, and to add or modify a description for it. Keep in mind that updating the name and description will also update the name and description of the corresponding object in the common asset service (CAS).

To edit a component, you can either open it directly from Structure view, or you can open a design that contains that component. You must use Design view to create a new component and to add a component to a design.

Viewing the thumbnail image for components

In Structure view, when you select a component, you can see the thumbnail image for that component on the General tab in the Properties panel. This thumbnail view provides an easy way to see the latest saved layout of the selected component without having to open it in Design view.

The thumbnail image reflects the layout of the component as it is visible in a design when it was last saved. When you create a component in a design and then exit Design view, the thumbnail image for that component is immediately available in Structure view. Unlike for designs, you do not need to first open the component directly in Design view .

Double-click the thumbnail image to open the selected component in Design view.

Viewing advanced properties for a component

In Structure view, you can select a component and see additional information about it that is stored in the common asset service (CAS).

To see these advanced details for a selected component, click the Advanced button on the General tab in the Properties panel. The Component details dialog box opens and displays the following information:

  • Name—The name of the component object in CAS
  • Version—The current version of the component. The version number is incremented every time that a new draft is created from an approved version of the component.
  • Description—The description of the component, if available
  • Created—The OTDS user that created the component, and the date and time that it was created
  • Last updated—The OTDS user that last updated the component, and the date and time that it was updated
  • Reserved by—The OTDS user that has reserved the component. This information is available only if the selected component is locked for editing.
Viewing references for a component

You can use the Used by tab in the Properties panel to see which designs use a selected component.

To find where a component is used:

  1. In the Structure map, select the component.

  2. In the Properties panel, click the Used by tab.

    The list shows all of the designs that use that component.

  3. Use the indicator in the Status column to determine whether the parent design is available for editing:

    • A green indicator () means that the parent design is available for editing.
    • A red indicator () means that the parent design is locked for editing by another user.
