Creating a Content Author template

To enable business users to add content, you must create a Content Author template that business users can use to create a Content Author theme (also called a communication) in Content Launcher. The Content Author theme is based on the design layout that you created in Communications Designer.

Content Author themes support only the page and email design types. Any web designs that are present in the communication are ignored when you create a Content Author template.

Creating a Content Author template

Important: You must approve and publish your communication, including all of the resources that are used in the communication, before you can create a Content Author template.

To create the Content Author template:

  1. In Structure view, in the Structure map, select Communication.

  2. In the Properties panel, on the Template tab, click the Create template button .

The Content Author template is created, and the template name is displayed in the Properties panel. The Content Author template has the same name as your communication, and the following information about the template is displayed in the template details area below the template name:

  • Version—The current version of the Content Author template. The version number is incremented every time that you update the template.
  • Update details—The OTDS user that updated the template and the date and time that it was updated
  • Associations—The Associated themes button that you can click to see the Content Author themes that are associated with each version of the template.

In Content Launcher, the newly created template is available in the list of templates in the Content Author widget. Users can use a single Content Author template to create multiple themes. For information about using Content Author, see OpenText Exstream Content Author Online Help.

Updating a Content Author template

To make an updated communication available in Content Author, you must update the Content Author template when you make any of the following changes to your communication:

  • Change the name of the communication
  • Create a new draft version of the communication
  • Update the associated design pack
  • Add or remove a document or design in the communication structure
  • Add, delete, or modify the frames in your designs

Important: When you update a template, the themes that are associated with the previous version of the template are not automatically associated with the updated version. To associate an existing theme with the updated version of the template, use Workshop to update your theme so that it references the new version of the template. For more information, see OpenText Exstream Workshop User Guide.

To update the Content Author template:

  1. In Structure view, in the Structure map, select the Communication button.

  2. In the Properties panel, on the Template tab, click Update.

    Tip: The Update button is visible only if your communication has been modified and the template needs to be updated because of that modification.

A new version of the Content Author template is created, and the following information about the template is displayed in the template details area below the template name:

  • Version—The current version of the Content Author template. The version number is incremented every time that you update the template.
  • Update details—The OTDS user that updated the template, and the date and time that it was updated
  • Associations—The Associated themes button that you can click to see the Content Author themes that are associated with each version of the template
