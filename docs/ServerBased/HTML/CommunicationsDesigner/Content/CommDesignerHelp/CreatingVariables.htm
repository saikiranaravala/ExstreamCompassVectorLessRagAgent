Creating variables

When you map a data source, you can use variables that are included in the associated design pack, or variables that you create in the Mapping view of the data source editor.

Variables that are created in the data source editor appear in italic in the Variables panel. This distinguishes them from variables that have been imported from a design pack (which are displayed as non-italic text).

After you create a variable, you can use the Variable properties section in the Properties panel to view and modify its properties to further define it.

Note: You cannot change the type of a variable after it is created.

When you create a variable in the data source editor, it then becomes available to all data sources within Communications Designer. If you change the variable properties, the changes carry over to all instances of the variable.

Once a variable is mapped to a data source, it is then available to any communication that is associated with that data source.

Creating a new variable

You can either create a new variable from the toolbar, or you can create and map a new variable to a specific data area.

  1. To create a new variable, select one of the following options:

    • On the toolbar, click the New variable button .

    • Point to the data area that you want to map, then click the Add new mapping button .

  2. In the New variable dialog, specify a name, and optionally, a description for the variable.

  3. In the Type list, select one of the following variable types:

    • String—Data that contains text

    • Integer—Data that is a whole number

    • Boolean—Data that must be one of only two possible values

    • Float—Data that is a numeric value that includes decimals

    • Date—Data that is a date, a time, or both

    • Currency—Data that is a monetary amount

    • Placeholder—Data that is content from an external source and that is accessed during an engine run. If you select this, you must also select the type of placeholder in the Placeholder type list.

  4. Click Create.

Deleting a variable

You can use Mapping view to delete only those variables that were created in the data source editor, and that are not currently used in any data sources or communications.

To delete a variable:

  1. In the Variables panel, select the variable.
  2. In the Variable properties panel, click the Delete button .

Tip: If the Delete button is not active, click the Used by card in the Variable properties panel to see if the selected variable is used in the current version of a data source or a communication object.

If no objects are listed in the properties panel, then the variable is used in a previous version of a data source or a communication object. Review the properties of the variable object in Workshop to determine where it is used. For more information about using Workshop, see the OpenText Exstream Workshop User Guide.
