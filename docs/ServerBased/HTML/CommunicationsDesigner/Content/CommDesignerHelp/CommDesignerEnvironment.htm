Communications Designer at a glance

When you first sign in, Communications Designer opens to the Communications page in Content Launcher.

Content Launcher is a common hub from which Exstream users can access both the Communications Designer design environment and the asset management environment to create, configure, and design your communications. The objects that are created and configured in Communications Designer are stored in the common asset service (CAS) and can be further managed using Workshop.

Note: Your OpenText Directory Services (OTDS) user role and the permissions that are assigned to that role determines which Communications Designer environments you can access. If you have questions about your access privileges, talk to your system administrator.

The following table describes how to use the Content Launcher toolbar to access the design and asset management environments:

 Environment

Communications button
