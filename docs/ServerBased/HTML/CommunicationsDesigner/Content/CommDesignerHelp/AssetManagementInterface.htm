Asset management environment

The asset management environment provides the user interfaces (or views) that you use to create, configure, and manage communication and design assets.

Note: Only certain Communications Designer roles allow access to the asset management environment. If you have questions about your access privileges, talk to your system administrator.

Communication asset management

On the Communication asset management page, you can create and edit data sources, use the properties panel to configure the properties for a selected data source, and then open the data source in Mapping view to map variables to specific data areas of a sample file.

In Mapping view, you can create variables, update variable properties, and change data area properties for mapped variables. You cannot access Mapping view from within the design environment. Mapping view contains the following user interface elements:

  • The toolbar lets you perform the following functions:

    • Hamburger menu ()—Access the help, sign out , display version information, or return to the Content Launcher

    • Save the data map

    • Close the data map

    • Undo or redo an action

    • Assign a customer start section

    • Create a new variable

    • Automap the sample file

  • The Variables panel displays the list of variables that are available to you. Use this panel to select the variable you want to map or edit. If you have a long list of variables, you can also categorize, narrow down, or search the variables list to find the variable that you need.

  • The Data Source Editor area gives you a visual representation of the sample file that is assigned to the data source. This is the panel where you will map variables to the data areas in the file.

    The Data Source Editor area also includes the Find element search bar, which is used to locate a specific data element within the sample file. Simply begin typing any part of the element name that you want to find, and any matching results are highlighted.

  • The properties panel is a dynamic panel that is always located on the right edge of Mapping view and displays different properties depending on the type of object that you have selected:

    • The Variable properties panel lets you view and modify the properties for a selected variable.

    • The Data area properties lets you view and modify how a variable is read for a specific data area. If the variable is assigned to multiple data areas, or is used in other data maps, changes to these properties will not affect how the variable is read in the other assigned data areas.

    • The Data source properties panel lets you change the encoding of the data source. This panel is displayed only when no variable is selected. All of the other information presented in this panel is not editable. To edit the other data source properties, use the properties panel on the Communication assets page.

Design asset management

On the Design asset management page, you can add fonts for use in your communications. It includes a properties panel, where you can view the properties for a selected font.
