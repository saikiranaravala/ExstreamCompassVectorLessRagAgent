Updating communication properties

As you create your communication structure, you can use the Properties panel to view and modify the communication properties.

In addition to the properties that are described in this section, you can also use Structure view to specify communication settings for Content Author templates and Empower output, and view the workflow history for the communication.

Updating basic properties

In the Properties panel, you can use the General tab to rename a communication, and to add or modify a description for it. Keep in mind that updating the name and description will also update the name and description of the corresponding object in the common asset service (CAS).

Viewing advanced details for a communication

In Structure view, you can select the communication and see additional information about it that is stored in the common asset service (CAS).

To see these advanced details for a selected communication, click the Advanced button on the General tab in the Properties panel. The Communication details dialog box opens and displays the following information:

  • Name—The name of the communication object in CAS
  • Version—The current version of the communication. The version number is incremented every time that a new draft is created from an approved version of the communication.
  • Description—The description of the communication, if available
  • Created—The OTDS user that created the communication, and the date and time that it was created
  • Last updated—The OTDS user that last updated the communication, and the date and time that it was updated
  • Reserved by—The OTDS user that has reserved the communication. This information is available only if the selected communication is locked for editing.
  • Design pack—The name, version, and CAS resource ID of the design pack that is associated with the communication
  • Data source—The name, version, and CAS resource ID of the data source that is associated with the communication. This information is available only if the communication uses a Communications Designer data source.
