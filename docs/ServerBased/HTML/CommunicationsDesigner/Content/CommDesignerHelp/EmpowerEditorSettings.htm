Specifying display settings for Empower Editor

In Communications Designer, if your communication includes multiple types of designs that contain interactive areas, you can choose which design is displayed by default when users open an Empower document.

Setting the default view for Empower Editor

When you first set up a communication, no default view is selected. If you do not set a default view for the communication, Empower Editor opens to the default view that is set in the Design Manager application that is associated with your communication.

To set the default view for Empower Editor:

  1. In Structure view, select the communication, and then click the Empower tab in the Properties panel.
  2. In the Default view list, select the design type that should appear as the default view in Empower Editor when end users open an Empower document.

Note: If your communication contains more than one email design and you select the Email design option, the first email design in the Structure map is displayed as the default view in Empower Editor.

Identifying interactive areas in Empower Editor

When theme view is enabled in Empower Editor, the colors in the following table indicate interactive areas in an Empower document so that end users can find them more easily:

Empower Editor interactive area indicators
Color Description
  Optional editable areas
  Optional editable areas that have been edited
  Required editable areas
  Required editable areas that have been edited

To switch between views in Empower Editor, users can click the Theme view button on the View Menu bar.
