A sample file is a JSON or XML file that contains the data areas and sample data that help you create data maps to identify the location of data and identify the start of a new customer.

Each data source is required to have a sample file, although the sample file can be changed at a later time.

The sample data is not the data source that the engine uses to produce applications, however. During production, the engine uses the data maps, along with the production data source, to read and write customer data.
