Automatically generated CSS styles

Based on the responsive features in your email and web designs, the Exstream engine automatically creates CSS styles for HTML-based output to define how the output should be displayed on a mobile and desktop devices.

Style placement for email designs

When you produce HTML (email) output from email designs, the engine places all of the automatically generated CSS styles within the generated *.html file by default. This is because email output must be a single, self-contained HTML file.

Style placement for web designs

When you produce HTML5 output from web designs, the placement of CSS styles is determined by the properties of the corresponding output object in the associated Design Manager application.

By default, the engine produces separate *.css files that are saved to the output directory and referenced by the generated *.html files. The engine creates these CSS files using the CSS3 standard. To include the CSS styles within the <head> element of the generated HTML files, you must update the output object properties. For more information about configuring output objects, see Creating Output in OpenText Exstream Design and Production Online Help.
