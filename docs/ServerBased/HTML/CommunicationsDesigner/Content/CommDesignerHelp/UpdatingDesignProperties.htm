Updating design properties

After you create a design in your communication structure, you can use the Properties panel to view and modify the design properties.

In addition to the properties that are described in this section, you can also use Structure view to specify flow and accessibility settings for page designs, and view the workflow history for each design.

Updating basic properties

In the Properties panel, you can use the General tab to rename a design, and to add or modify a description for it. For page designs, you can also update the paper type. Keep in mind that updating the name and description will also update the name and description of the corresponding object in the common asset service (CAS).

Viewing the thumbnail image for designs

In Structure view, when you select a design, you can see the thumbnail image for that design on the General tab in the Properties panel. This thumbnail view provides an easy way to see the latest saved layout of the selected design without having to open it in Design view.

The thumbnail image reflects the layout of the design as it is visible in Design view when it was last saved, and is available only after at least one user has opened the design in Design view. For example:

  • If you select the Variable sample data button on the design toolbar and then save a design, then the thumbnail will display the sample data instead of variable names.
  • For web and email designs, if you select the Mobile view button on the design toolbar and then save the design, then the thumbnail will display the mobile layout of the design.

Double-click the thumbnail image to open the selected design in Design view.

Viewing advanced properties for a design

In Structure view, you can select a design and see additional information about it that is stored in the common asset service (CAS).

To see these advanced details for a selected design, click the Advanced button on the General tab in the Properties panel. The Design details dialog box opens and displays the following information:

  • Name—The name of the design object in CAS
  • Version—The current version of the design. The version number is incremented every time that a new draft is created from an approved version of the design.
  • Description—The description of the design, if available
  • Created—The OTDS user that created the design, and the date and time that it was created
  • Last updated—The OTDS user that last updated the design, and the date and time that it was updated
  • Reserved by—The OTDS user that has reserved the design. This information is available only if the selected design is locked for editing.
Specifying targeting rules

Targeting rules determine whether a design is included in the output when you run the engine.

Important: Library rules and variables that are available for creating custom rules depend on the design pack and the data source (if any) that is associated with your communication. If you insert designs into your communication from other communications that use different design packs, targeting rules on the inserted designs might not function correctly.

Adding a new targeting rule

  1. On the Targeting tab, in the Design rule area, click Add.

  2. In the Rule Composer, select the type of rule that you want to apply:

    • Library rule—Use this tab to select a rule from the list of available rules. You can also use the Search box to filter the list of rules.
    • Custom rule—Use this tab to define a custom targeting rule using the variables that are available in that communication.
Updating an existing targeting rule

  1. On the Targeting tab, in the Design rule area, click Edit.

  2. In the Rule Composer, you can choose to modify or remove the existing rule:

    • To modify the rule, make any needed changes, then click Insert.
    • To remove the rule, click Remove.
Viewing document references for a design

You can use the Used by tab in the Properties panel to see which documents use a selected design.

To find where a design is used:

  1. In the Structure map, select the design.

  2. In the Properties panel, click the Used by tab.

    The list shows all of the documents that use that design.

  3. Use the indicator in the Status column to determine whether the parent document is available for editing:

    • A green indicator () means that the parent document is available for editing.
    • A red indicator () means that the parent document is locked for editing by another user.
