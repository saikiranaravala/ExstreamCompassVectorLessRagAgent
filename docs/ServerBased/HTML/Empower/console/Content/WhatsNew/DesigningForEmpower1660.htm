Empower design features

  • Show/hide functionality for selection groups
  • Support to edit automated table rows in Empower
  • Add interactive areas in Communications Designer for Empower users
  • Content Author support for interactive properties on inline editable areas
  • Support for search keys through fulfillment
  • Display adjusted table cells in Empower designs

Show/hide functionality for selection groups

Designers can now allow add selection groups to allow ends users to make selections to indicate which text paragraphs or paragraph objects should be visible in their Empower document.

For more information about using selection groups in Empower, see .

Support to edit automated table rows in Empower

Designers can now allow end users to add, duplicate, or remove automated rows in tables in Empower Editor.

Note: The minimum version of the Exstream engine that is required to support this functionality is 16.6.0.

For more information about editing automated tables in Empower Editor, see .

Add interactive areas in Communications Designer for Empower users

To streamline business processes that support customer interactions, Communications Designer now supports adding interactive areas to designs that can then be edited in Empower Editor.

In Design View, you can specify interactive properties on the following objects:

  • Text boxes
  • Embedded text boxes
  • Inline text within text boxes
  • Variables

To enable interactive areas in customer output, you must include an Empower output queue in the Design Manager application that you initially create and package for use with Communications Designer.

For information about adding output queues to an application, see .

Content Author support for interactive properties on inline editable areas

Business users in Content Author can now add interactive properties to inline areas that will be edited in Empower Editor.

The options that are available for inline variables in clauses are the same as other interactive elements—for example, making changes to content or formatting, adding tooltips, and specifying a list of options for end users in Empower Editor to select from.

For more information about configuring content to be editable in Empower Editor, see the Content Author Help, which is available from the hamburger menu in Content Author.

Support for search keys through fulfillment

Designers can now place search keys in an Empower fulfillment application to pass through to fulfillment, which will enable them to provide embedded instructions to end users.

For more information about using search keys in your application, see .

Display adjusted table cells in Empower designs

Empower is now able to correctly display table cells that are designed with a different width than the column in which they are placed.
