New features in version 16.6.0

This release of Exstream Empower Editor adds new functionality, and also includes improvements to documentation and bug fixes.

The following table provides a summary of the new features and functionality changes in this release. For a list of deprecated or discontinued features, see Discontinued and deprecated features and platforms.

New and updated features in version 16.6.0
Category What's new
Empower design
  • Show/hide functionality for selection groups
  • Support to edit automated table rows in Empower
  • Add interactive areas in Communications Designer for Empower users
  • Content Author support for interactive properties on inline editable areas
  • Support for search keys through fulfillment
  • Display adjusted table cells in Empower designs
Empower Editor
  • Customizing the appearance of Empower Editor
