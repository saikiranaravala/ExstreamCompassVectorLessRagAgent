Customizing the appearance of Empower Editor

New options have been added to the editor customization features that were introduced in version 16.4 Update 2 of Empower. In addition to the existing options to customize the logo and the title, menu, and tool bars in Empower Editor, Empower administrators can now also specify custom colors for the Preview button and the Page Manager controls.

To use the Empower Server Console to customize the editor interface, simply click EDITORS > CUSTOMIZE in the server console, and then upload a logo or adjust the color settings as needed in the customization settings dialog box.

You can also use the following Empower API calls to view and set the color properties of the editor interface:

  • Get editor skin properties—Returns a list of the current values for the configurable color properties that can be used to customize the appearance of Empower Editor
  • Set editor skin properties—Edits the configurable color properties that can be used to customize the appearance of Empower Editor

Keep in mind that editor customization is not supported in editor versions earlier than 16.4 Update 2, and some customization options are available only in version 16.6.0 and later. Custom settings will be applied only to the editors that support each option.

For information about using the Empower Server Console or the Empower API to customize the appearance of Empower Editor, see the help that is available from within the server console.
