Editors

Every Empower application is configured to use a specific version of Empower Editor to open the documents that are associated with that application. Empower lets you manage multiple versions of the editor concurrently, and to add newer versions of the editor as they become available, in order to take advantage of new and updated features.

In this section

  • Adding a new version of the editor
  • Assigning a different editor to an application
  • Assigning the default editor to all applications
  • Changing the editor used by a document
  • Customizing the appearance of the editor
  • Uninstalling a version of the editor
