Uploading a document

  1. In the Empower Server Console, click DOCUMENTS.

  2. On the Documents page, click UPLOAD.

  3. In the DOCUMENT UPLOAD dialog box, you can optionally enter your own business-defined document identifier in the External document ID box.

    You can use this identifier to search for your document on the Documents page, however, keep in mind that it does not replace the unique document ID that Empower generates for every Empower document. The external document ID cannot be changed after the document is uploaded.

  4. If you want to associate the document with a specific Empower application, enter the ID of that application in the Application ID box.

    You are not required to provide an application ID because Empower automatically assigns the document to the Empower application that matches a specific set of document attributes (the Application Object Identifier (OI), the application path, the database ID, and the package creation date).

    Note: Keep in mind that specifying an Empower application ID that does not match the document can produce unexpected results in a production environment, and should be used for testing purposes only.

  5. In the Document owners box, add a unique identifier for each user that you want to add as a document owner (optional).

    The identifier that you enter (such as a user ID or an email address) must match the identifier that your authorization provider passes to Empower for the user.

    If you do not specify a document owner, then all authenticated users are permitted to open and edit the document.

    Note: If access to an Empower document is restricted by access groups (which are inherited from the Empower application used to create the document), then each user that you add as a document owner must be a member of an access group that is permitted to open and edit the document.

  6. In the User-defined tags box, enter one or more tags for your document (optional).

  7. Click DOCUMENT, and use the file selection dialog box to choose the Empower document that you want to use to create the application.

  8. Click UPLOAD.
