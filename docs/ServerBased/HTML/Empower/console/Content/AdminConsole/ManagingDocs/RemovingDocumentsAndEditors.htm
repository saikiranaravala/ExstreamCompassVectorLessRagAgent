Removing documents and editors

Empower comes with an automated cleanup process that by default runs once a week to remove unused documents and deactivated editors. If you need the cleanup process to run more frequently, you can use the Empower Server Console to customize the cleanup schedule, so that it runs on certain days or at specific intervals.

Note: If your workflow includes a process that uses an API call to set the cleanup schedule, the cleanup schedule settings that are displayed in the Empower Server Console might not reflect those values. Keep in mind that using the Empower Server Console to change the cleanup schedule will override any values that were set previously by an API call that is part of another process.

Before you begin configuring the cleanup process, it is helpful to understand how the process works.

Depending on which options you choose, the following actions are performed when the cleanup process runs:

  1. The cleanup process searches for documents that have not been edited within a specified number of days, and places those documents in the trash (optional).

  2. The cleanup process checks for documents that have been retained in the trash for a specified number of days, and permanently deletes any documents that exceed the trash retention period.

  3. The cleanup process checks for editors that have been deactivated, and uninstalls deactivated editors that are no longer assigned to a document.

    Important: When you uninstall an editor that is assigned to one or more documents, Empower deactivates the editor, but does not remove it immediately. Instead, Empower retains the deactivated editor until the last document that uses the editor is permanently deleted. After the last document is permanently deleted, the scheduled cleanup process removes the deactivated editor. For information about uninstalling an editor, see Uninstalling a version of the editor

To minimize the impact of document repository changes that can occur during the cleanup process, schedule the cleanup process to run outside of business hours.

To configure the cleanup process:

  1. In the Empower Server Console, click CONFIGURATION.

  2. In the Cleanup schedule area, complete one of the following sets of steps:

    To Do this
    Schedule cleanup to run once a day on specific days of the week
    1. In the Frequency list, select Daily.
    2. In the Days section, select the days of the week on which you want the cleanup process to run. To run the cleanup process every day, select All.
    1. In the Start time boxes, specify when the cleanup process will begin. For example, if you select a start time of 11:00, the cleanup process will begin at 11:00 AM on the days that you selected. If you select a start time of 23:00, the cleanup process will begin at 11:00 PM on the days that you selected.
    Schedule cleanup to run daily at set intervals
    1. In the Frequency list, select Hourly.
    2. In the Interval (hours) box, specify how often you want to run cleanup. The cleanup process will start running at midnight, and then run on the interval you set. For example, if you select 4, the cleanup process will run every four hours.

  3. If you want Empower to search for documents that have not been edited within a specified number of days and add them to the trash when the cleanup process runs, select the Idle document cleanup check box.

    If you do not want documents to be automatically added to the trash when the scheduled cleanup process runs, make sure that this check box is cleared.

    For information about how to manually add documents to the trash, see Manually deleting documents.

  4. If you have enabled idle document cleanup, in the Idle document retention (days) box, specify the number of days that an idle document will be retained before it is added to the trash. An idle document is one that has not been edited during the retention period.

  5. In the Trash retention (days) box, specify the number of days that objects will be retained in the trash before being permanently removed from Empower.

  6. Click UPDATE.
