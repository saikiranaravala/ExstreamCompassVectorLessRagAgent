Replacing primary dictionaries

As of version 16.2, the following primary dictionaries are not included with Empower: Danish, German, Hebrew, Hindi, Indonesian, Italian, Malay, and Vietnamese. In order to have spell checking available in Empower Editor for any of the affected languages, you will need to manually add Hunspell dictionary files to the Empower Server.

For information about how to obtain or create Hunspell dictionary files, refer to the Hunspell documentation on the SourceForge website.

To replace primary dictionaries:

  1. In the application server that hosts Empower, locate the folder where the Empower primary dictionaries are stored. For example, the folder path for TomEE+ in a Windows environment is CATALINA_HOME\webapps\empower<xx.x.xxxx>\dictionaries.
  2. Within the dictionaries folder, create a subfolder for each of the primary dictionaries that you need to add:

    For this languageCreate this subfolder
    Danish

    da_DK

    Germande_DE
    Hebrewhe_IL
    Hindihi
    Indonesianid_ID
    Italianit_IT
    Malayms_MY
    Vietnamesevi_VN

    Important: Empower supports replacing only the primary dictionaries in this list. You cannot add primary dictionaries that are not included in the installation package or in this list.

  3. Place the appropriate Hunspell dictionary files inside each of the subfolders that you created.
  4. Restart your application server.
