Managing tags for end user documents

To add or change tags in the Empower Server Console:

  1. In the Empower Server Console, click DOCUMENTS.

  2. On the Documents page, use the sorting, filtering, or search tools to locate the document that you want to edit.

    Note: You can only edit active documents. For information about restoring a document that is no longer active, see Restoring documents from the trash .

  3. To open the details page for the document that you want to edit, click the document ID.

  4. On the Document Details page, in the User-defined tags box, add a new tag or remove an existing tag.

    Note: User-defined tags cannot exceed 250 characters in length and can contain only letters, numbers, periods (.), hyphens (-), underscores (_), and spaces. If you enter an invalid character, the User-defined tags field turns pink and the UPDATE button becomes inactive.

  5. Click UPDATE.

    To return to the Documents page, click Back to Documents.
