Managing dictionaries for spell checking

Important: As of version 16.2, the following primary dictionaries are not included with Empower:

  • Danish (da_DK)

  • German (de_DE)

  • Hebrew (he_IL)

  • Hindi (hi)

  • Indonesian (id_ID)

  • Italian (it_IT)

  • Malay (ms_MY)

  • Vietnamese (vi_VN)

    • For information about how to replace these primary dictionaries, see Replacing primary dictionaries.

Empower comes with a set of language-specific dictionaries that it uses for spell checking Empower documents in Empower Editor (the dictionary language is determined by the document settings). Empower also supports custom dictionaries and excluded words that are passed to Empower from applications that were created in Exstream Design and Production. Sometimes, however, words are flagged as being misspelled when they should not be. This issue typically occurs when a document contains words that do not appear in a standard dictionary, such as medical terms or proper names. To add support for words that are not included in the standard dictionary, Empower administrators can upload a language-specific Hunspell dictionary file, and map it to an existing primary dictionary as an additional resource.

For information about how to obtain or create Hunspell dictionary files, refer to the Hunspell documentation on the SourceForge website.

To manage the spell-checking dictionaries in Empower, complete the following tasks as needed:

  • Changing the primary dictionary
  • Uploading a secondary dictionary
  • Removing a secondary dictionary

Changing the primary dictionary

  1. In the Empower Server Console, click DICTIONARIES.

  2. In the Language list, select the language of the dictionary that you want to use.

    The Primary field is updated with the name of the dictionary that Empower uses for the language you selected.

Uploading a secondary dictionary

Note that Empower supports only a single primary dictionary and a single secondary dictionary for each language. If a secondary dictionary has already been chosen and you upload another secondary dictionary file, then the dictionary file that you upload replaces the existing secondary dictionary.

To upload a secondary dictionary:

  1. In the Empower Server Console, click DICTIONARIES.

  2. In the Language list, select the language of the dictionary that you want to configure (if it is not already selected).

    Note: You can change only the secondary dictionary.

  3. Click UPLOAD.

  4. Use the file selection dialog box to locate and choose the Hunspell dictionary file that you want to use for the secondary dictionary. Hunspell dictionary files have a .dic extension.

    After the dictionary file is uploaded, the Secondary field is updated with the name of the dictionary, indicating that the new dictionary has been saved.

Removing a secondary dictionary

  1. In the Empower Server Console, click DICTIONARIES.

  2. In the Language list, select the language of the secondary dictionary that you want to remove (if it is not already selected).

    Note: You can remove only the secondary dictionary.

  3. Click REMOVE.

  4. Confirm that you want to delete the secondary dictionary for the selected language.

    The Secondary field is cleared, indicating that the dictionary has been removed from Empower.
