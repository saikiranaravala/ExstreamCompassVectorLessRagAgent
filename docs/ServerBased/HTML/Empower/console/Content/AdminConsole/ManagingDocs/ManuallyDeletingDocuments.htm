Manually deleting documents

Empower comes with an automated cleanup process that periodically removes unused documents. However, there are times when you as an administrator might find it necessary to manually delete documents from Empower (for example, if a document is uploaded by mistake and is not subsequently exported). In this case, you can use the Empower Server Console to delete documents and place them in the trash.

To manually delete documents from Empower:

  1. In the Empower Server Console, click DOCUMENTS.

  2. To view only documents that are active, select Active from the drop-down list.

  3. To sort the list of documents according to the last date and time that they were saved, click the Last saved table heading.

  4. Select the check box next to each document that you want to add to the trash, and then click DELETE.

  5. Confirm that you want to delete the selected documents.

    The dialog box closes and the documents that you selected are added to the trash.

    Depending on how the automated cleanup process is configured, documents that you add to the trash might be held in the trash for a period of time before being permanently removed by the scheduled cleanup process. For more information about the automated cleanup process, see Removing documents and editors.
