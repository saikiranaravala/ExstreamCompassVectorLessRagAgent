User documentation

Empower user documentation is available from within the software as well as on My Support at https://knowledge.opentext.com/go/ExstreamDocumentation (go to latest version > Interactive > Empower).

The online documentation that can be accessed from the software is hosted on the OpenText Global Help Server and provides users with live access to the latest version of the documentation.

You can also deploy and OpenText Private Help Server instance to provide local access to downloaded documentation. The Private Help Server lets users access the online documentation within their private corporate network. After you set up the server, you can then redirect help requests from Exstream Design and Production software to the Private Help Server instance rather than the Global Help Server. For information about setting up the Private Help Server for you Empower documentation, see the Empower installation guide for your application server type.

The downloadable HTML and PDF versions of the Exstream Design and Production documentation are available as zipped archives in My Support.

Empower documentation

  • For an overview of the latest version of Empower, including new features, delivery information, and supported platforms, see the Exstream Empower Editor Release Notes, which you can download at https://knowledge.opentext.com/go/ExstreamDocumentation (latest version > Interactive > Empower).

  • For information about how to install or upgrade Empower, download the version of Installing Empower that corresponds to your application server type at https://knowledge.opentext.com/go/ExstreamDocumentation (latest version > Interactive > Empower).

  • For information about using the Empower Server Console to manage the Empower Services Layer and move documents in and out of the Empower Services Layer, click GET HELP > Documentation (English) in the Empower Server Console.

  • For information about using the Empower API to manage the Empower Services Layer and move documents in and out of the Empower Services Layer, click GET HELP > Documentation (English) in the Empower Server Console.

  • Exstream designers can access the Designing for Empower Editor guide as an integrated part of the HTML5 Exstream Design and Production documentation set. To access the documentation, click Help > Search Documentation on the Menu bar in either Design Manager or Designer.

  • End users of the Empower Editor browser-based document editing tool can access user help and documentation by clicking HELP > ? within the editor interface.

Related documentation

For information about Exstream Design and Production, or for supplemental information about related products, see the Exstream documentation at https://knowledge.opentext.com/go/ExstreamDocumentation.
