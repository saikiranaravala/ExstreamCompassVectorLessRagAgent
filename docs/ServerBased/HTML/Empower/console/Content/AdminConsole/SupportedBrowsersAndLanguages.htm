Supported browsers and languages

This section describes web browser and language support for the various components of Empower.

Supported web browsers

The Empower Server Console, Empower Editor, and related help can be opened in the following web browsers:

  • Google Chrome, latest stable release

  • Microsoft Edge, latest stable release

  • Microsoft Internet Explorer, version 11 (Enterprise Mode not supported)

  • Mozilla Firefox, latest stable release

Available languages

  • Release notes—The release notes are available in English.

  • Installation and upgrade guides—The installation and upgrade guides are available in English.

  • Empower Server Console—The server console and the console online help are available in English.

  • Empower API—The API and the API documentation are available in English. The API documentation is part of the online help in the Empower Server Console.

  • Help for Empower designers—Designing for Exstream Empower Editor is available in English. Localized versions of this help will be available in April 2019. You can access Designing for Exstream Empower Editor through the online help in Exstream Designer or Exstream Design Manager.

  • Empower Editor—The editor interface and the editor online help are available in the following languages:

    • Brazilian Portuguese
    • Chinese
    • Dutch
    • English
    • French
    • German
    • Japanese
    • Spanish

    If your web browser is configured to use one of the languages that Empower supports, Empower Editor derives its language settings from the browser. However, if your browser is not configured to use one of the supported languages, then Empower Editor uses the default language that the Empower administrator has specified for your server.
