Deleting applications

As the repository grows, an important part of administering the Empower system will be making sure that unneeded Empower applications are deleted on a regular basis. Empower comes with an automated cleanup process that periodically removes unused documents and deactivated editors; however, this process does not remove applications. Therefore, you might find it necessary to periodically manually remove some applications, to reduce the amount of time that it takes Empower to search for applications.

To delete applications from Empower:

  1. In the Empower Server Console, click APPLICATIONS.

  2. On the Applications page, use the sorting, filtering, or search tools to locate the applications that you want to delete.

  3. Select the check box next to each application that you want to remove, and then click DELETE.

  4. Confirm that you want to delete the selected applications.

    Note: Keep in mind that if you delete an Empower application that is assigned to a document, users are still able to edit and delete the document. However, if a user attempts to upload the document again, Empower will notify the user that the document cannot be uploaded until a matching application is added. If a user adds an application that has the same attributes as an application that was previously deleted, Empower assigns the ID of the deleted application to the newly-added application. Additionally, any documents that were associated with the ID of the deleted application are now associated with the new Empower application.
