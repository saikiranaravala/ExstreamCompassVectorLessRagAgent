Changing the editor used by a document

In some cases you might find it necessary to change the version of the editor that a document uses; for example, if you upgrade Empower to use a later version of Empower Editor. To change the version of the editor that a document uses, you must assign a different version of the editor to the Empower application that is associated with the document, and then upload the document again.

To assign a different version of the editor to the Empower application that is associated with a document:

  1. In the Empower Server Console, click DOCUMENTS.

  2. On the Documents page, click the ID of the document that you want to change.

  3. On the Document Details page, note the Application ID of the document.

  4. Click APPLICATIONS.

  5. On the Applications page, click the ID of the Empower application that you want to change.

  6. From the Editor version drop-down list, select the version of the editor that you want to use.

  7. Click UPDATE.

  8. Export the document (optional). You should export a document only if it is no longer being used.

    For information about exporting a document, see Exporting a document.

  9. Upload the document.

    The uploaded document is assigned the version of the editor that is saved in the Empower application. For more information about uploading a document, see Uploading a document.
