Assigning the default editor to all applications

In some cases, you might find it useful to be able to update all of your existing Empower applications at one time to use the default version of Empower Editor. For example, in environments that routinely support numerous Empower applications, this functionality can help to streamline the process of updating your applications when a newer version of the editor is added to Empower.

When you use the UPDATE ALL functionality to assign the default version of the editor to your existing applications, there are a couple of things to keep in mind:

  • The editor version will not change for existing Empower documents.
  • New Empower documents that reference one of the updated applications will use the default version of the editor.

To assign the default version of Empower Editor to all applications:

  1. From the Empower Server Console, click EDITORS.

  2. Make sure that the version of the editor you want to assign to all applications is marked as the Default editor. If it is not, then select the version that you want to use and click SET DEFAULT.

  3. Next to the default version of the editor, click UPDATE ALL, and then confirm that you want to make the change.

To verify that your applications have been updated to use the default editor, click APPLICATIONS, and then check the Editor version column of the applications table.
