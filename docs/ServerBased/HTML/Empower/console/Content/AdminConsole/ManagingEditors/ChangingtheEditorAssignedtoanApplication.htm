Assigning a different editor to an application

Every Empower application is configured to use a specific version of Empower Editor to open the documents that are associated with that application.

When you add an application to Empower, if you do not specify a version of the editor for the application to use, Empower assigns the system default version of the editor to the application.

To assign a different version of Empower Editor to an application:

  1. From the Empower Server Console, click APPLICATIONS.

  2. On the Applications page, use the sorting, filtering, or search tools to locate the application that you want to edit, and then click the ID of the application.

  3. On the Application Details page, in the Editor version list, select the version of the editor that you want to assign to the application.

  4. Click UPDATE.

Tip: To assign an inactive editor to an Empower application and its new documents, you must first go to the EDITORS tab of the console and activate the version of the editor that you want to use.

Viewing the applications associated with a specific version of the editor

  1. From the Empower Server Console, click APPLICATIONS.

  2. In the search ribbon, use the drop-down list to select the version of the editor that you want to use to filter the list of applications.
