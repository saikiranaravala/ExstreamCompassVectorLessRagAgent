Customizing the appearance of the editor

Optionally, you can customize the appearance of the Empower Editor interface that is displayed to end users, by choosing the colors to use for some parts of the interface, or by adding a custom logo to the interface.

Important: Editor customization is not supported in editor versions earlier than 16.4 Update 2, and some customization options are available only in version 16.6.0 and later. Custom settings will be applied only to the editors that support each option.

To customize the appearance of Empower Editor:

  1. From the Empower Server Console, click EDITORS.

  2. On the EDITORS page, click CUSTOMIZE.

  3. In the EDITOR CUSTOMIZATION SETTINGS dialog box, configure the color and logo settings as needed. The color settings support only hexadecimal color values. If a full six-digit color value is not used, the incomplete color code will be padded with zeroes.

    Settings for Empower 16.4 Update 2 and later:

    Title bar
    • Title bar color—Enter the color to use for the title bar.
    • Text color—Enter the color to use for the title bar text.
    Menu bar and toolbar
    • Menu bar first gradient—Enter the first color to use for the menu bar gradient.
    • Menu bar second gradient—Enter the second color to use for the menu bar gradient.
    • Active tab color—Enter the color to use for highlighting the active selection on the menu bar.
    • Hover color—Enter the color to use for emphasizing a menu item that an end user points to.
    • Text color—Enter the color to use for the menu item text.
    • Toolbar color—Enter the color to use for the background of the toolbar.
    Logo

    To add a custom logo, click UPLOAD, and choose the image that you want to use. When choosing an image file, keep in mind the following considerations:

    • Images must be in PNG format.
    • The logo will be displayed at a height of 28 pixels.
    • Images with an aspect ratio greater than 20:1 will be cropped.

    Settings for Empower 16.6.0 and later:

    Preview button
    • Button color—Enter the color to use for the Preview button.
    • Hover color—Enter the color to use for highlightinging the Preview button when an end user points to it.
    • Text color—Enter the color to use for the Preview button text.
    Page Manager
    • Background color—Enter the color to use for the background of the Page Manager selection box.
    • Page Manager text color—Enter the color to use for the text in the Page Manager selection box.
    • Border hover color—Enter the color to use for emphasizing the border of the Page Manager selection box when an end user points to it.
    • Paging button hover color—Enter the color to use for emphasizing the pagination controls when an end user points to them.

  4. Click OK.

    Your changes will be applied to all editors that support each option.
