Uninstalling a version of the editor

Because Empower regularly releases new versions of the editor, you might want to remove older versions of the editor that are no longer being used.

To uninstall a version of the editor:

  1. In the Empower Server Console, click EDITORS.

  2. On the Editors page, select the check box next to the version of the editor that you want to uninstall and then click UNINSTALL.

    Empower does one of the following:

    If Empower does this
    The editor is the default system editor Takes no action
    The editor is assigned to an Empower application Takes no action
    The editor is not assigned to an Empower application, but is used by one or more documents Deactivates the editor, which means that the editor cannot be assigned to any Empower applications. The deactivated editor will be uninstalled (permanently removed) after the last document that uses that version of the editor is deleted. For more information, see Manually deleting documents.
    The editor is not assigned to an Empower application or used by any documents Uninstalls the editor, which means that it is immediately and permanently removed from Empower
