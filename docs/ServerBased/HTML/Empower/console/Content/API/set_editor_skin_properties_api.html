Set editor skin properties

The POST API call for editor skin properties lets you customize the appearance of Empower Editor by choosing some of the colors for the interface that is displayed to Empower end users.

Important: Editor customization is not supported in editor versions earlier than 16.4 Update 2, and some customization options are available only in version 16.6.0 and later. Custom settings will be applied only to the editors that support each option.

Who can access this call

Access to the Empower API is defined in the customSecurityContext.xml file during installation, and is controlled by your authorization provider. By default, Empower supports the administrator, editor, and integrator authorities.

Authenticated users with the following authority can use this API call:

  • administrator

Request

Request method Content type Syntax
POST application/x-www-form-urlencoded /<context>/resource/skin

The <context> variable is configured during Empower installation.

Request header

Header Value Use Description
X-CSRF-TOKEN {csrfToken} Required

Provides a session token that authorizes you to access the DELETE and POST API calls

To view the token for your current session, do one of the following:

  • To capture the csrfToken response parameter, see Get security token.
  • Retrieve the XSRF-TOKEN cookie after making an API call from a session that is already in progress.

Keep in mind that if a session expires or changes, you must acquire a new security token.
Accept application/xml Optional Requests that Empower return an XML response instead of the default JSON response

Request parameters

Parameters that can be used with Empower 16.4 Update 2 and later:

Parameter Data type Use Description
menubarActiveTabColor String
 Optional

The color to use for highlighting the active selection on the menu bar
menubarBottomBackground String Optional

The second color to use for the menu bar gradient
menubarHoverTabColor String Optional

The color to use for emphasizing a menu item that an end user points to
menubarTextColor String Optional

The color to use for the text of menu items
menubarTopBackground String Optional

The first color to use for the menu bar gradient
titlebarBackground

String

 Optional

The color to use for the title bar
titlebarText

String

 Optional

The color to use for the title bar text
toolbarBackground String Optional

The color to use for the background of the toolbar
These parameters support only hexadecimal color values (3 or 6 characters accepted).

Parameters that can be used with Empower 16.6.0 and later:

Parameter Data type Use Description
pageButtonHover String Optional

The color to use for emphasizing the pagination controls when an end user points to them
pageManagerBackground String Optional

The color to use for the background of the Page Manager selection box
pageManagerHoverBorder String Optional

The color to use for emphasizing the border of the Page Manager selection box when an end user points to it
pageManagerText String Optional

The color to use for the text in the Page Manager selection box
previewButton String Optional

The color to use for the Preview button
previewButtonHover String Optional

The color to use for emphasizing the Preview button when an end user points to it
previewButtonText String Optional

The color to use for the Preview button text
These parameters support only hexadecimal color values (3 or 6 characters accepted).

Example request

Element Example

Request header

 POST http://sample-server.example.com:8080/mpw/resource/skin
X-CSRF-TOKEN: 95cf86a3-079f-4860-aa62-dd64f8a8c321
Request body Content-Type: application/x-www-form-urlencoded;

Content-Disposition: form-data;
titlebarBackground=#000002&titlebarText=#000001&menubarTopBackground=#000005&menubarTextColor=#000007
