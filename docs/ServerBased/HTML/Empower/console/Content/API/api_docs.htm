Empower API documentation

The Empower API is a REST interface that lets you use API calls to control functions such as importing documents to Empower Editor, opening documents in the editor interface, and exporting documents for fulfillment.

You can use the Empower API calls in your integration application to perform operations on Empower access groups, applications, documents, editors, and system settings.

Access groups

API call Description
Add an access group Adds a new access group
Delete an access group Deletes a specified access group
Get access groups Returns a list of the access groups that match your request parameters
Update an access group Modifies the attributes of an access group

Applications

API call Description
Batch delete applications Deletes multiple applications
Batch update applications Assigns a new version of the editor to multiple applications
Create an application Creates an application
Delete an application Deletes a specified application
Get applications Returns a list of the applications that match your request parameters
Update all applications to use the default editor Updates all applications to use the default version of Empower Editor
Update an application Modifies the attributes of an application

Documents

API call Description
Batch delete documents Marks multiple documents for deletion
Batch restore documents Restores multiple documents
Delete a document Marks a specified document for deletion
Export a document Returns the specified Empower document in the body of the response, and then marks the document for deletion
Get documents Returns a list of the documents that match your request parameters
Get document metadata Retrieves the metadata for a specified document
Import a document Imports an Empower document and assigns it to an existing Empower application
Open a document Opens a specified Empower document
Restore a document Restores a specified document
Update a document Modifies the attributes of a document

Editors

API call Description
Get editors Returns a list of the editors that match your request parameters
Reactivate an editor Reactivates a specified version of the editor
Uninstall an editor Uninstalls or deactivates a specified version of the editor
Update all applications to use the default editor Updates all applications to use the default version of Empower Editor
Upload an editor Installs a new version of the editor interface

Editor customization

API call Description
Delete editor custom logo

Removes the custom logo image that was uploaded previously to Empower Editor
Get editor custom logo Returns the binary content of the file being used to provide the custom logo image in Empower Editor
Get editor skin properties Returns a list of the current values for the configurable color properties that can be used to customize the appearance of Empower Editor
Set editor custom logo Adds a custom logo image to Empower Editor
Set editor skin properties Edits the configurable color properties that can be used to customize the appearance of Empower Editor

System settings

API call Description
Get a configuration Returns a specified system configuration setting
Get all configurations Returns all system configuration settings
Get dictionary mappings Returns a list of all of the Empower primary dictionaries (by language), along with the name of the secondary dictionary that is mapped to each primary dictionary
Get security token Returns information about the security token that is associated with your current session
Get users

Returns a list of Empower users who have opened a document in Empower Editor within a specified number of days, with the date and time of the most recent sign in for each user on the list.
Import a secondary dictionary Imports a Hunspell dictionary file to the Empower Services Layer and maps it to an existing primary dictionary as an additional resource
Run cleanup scheduler Runs the cleanup process on demand
Set configurations Sets the value of one or more system configuration attributes
System healthcheck Returns the status of the components of the Empower system
View response codes Returns a file that contains the Empower API response messages and their associated codes
