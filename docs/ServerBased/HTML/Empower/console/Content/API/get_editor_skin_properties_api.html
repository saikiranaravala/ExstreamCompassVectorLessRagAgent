Get editor skin properties

The GET API call for editor skin properties returns a list of the current values of the configurable color properties that can be used to customize the appearance of Empower Editor.

Important: Editor customization is not supported in editor versions earlier than 16.4 Update 2, and some customization parameters are available only in version 16.6.0 and later.

For information about which parameters are supported in each version, see Set editor skin properties.

Who can access this call

Access to the Empower API is defined in the customSecurityContext.xml file during installation, and is controlled by your authorization provider. By default, Empower supports the administrator, editor, and integrator authorities.

Authenticated users with any of the following authorities can use this API call:

  • administrator
  • editor

Request

Request method Syntax
GET /<context>/resource/skin

The <context> variable is configured during Empower installation.

Request header

Header Value Use Description
Accept application/xml Optional Requests that Empower return an XML response instead of the default JSON response

Request parameters

This call does not require parameters.

Example request

Element Example

Request header

 GET http://sample-server.example.com:8080/mpw/resource/skin
Request body This API call does not support a request body.

Response

Empower supports two formats for API response payloads: JSON and XML.

    </body>

Response parameters

If a parameter object does not contain a value, a JSON response returns a null value for the object, but an XML response does not include the object.

Parameter Data type Description
logoFilename String The name of the custom logo image file
logoMimetype String The MIME type of the custom logo image file
menubarActiveTabColor String

The color that is used to highlight the active selection on the menu bar
menubarBottomBackground String

The second color that is used to create the menu bar gradient
menubarHoverTabColor String

The color that is used to emphasize a menu item that an end user points to
menubarTextColor String

The color of the menu item text
menubarTopBackground String

The first color that is used to create the menu bar gradient
pageButtonHover String

The color that is used to emphasize the pagination controls when an end user points to them
pageManagerBackground String

The color of the background of the Page Manager selection box
pageManagerHoverBorder String

The color that is used to emphasize the border of the Page Manager selection box when an end user points to it
pageManagerText String

The color of the text in the Page Manager selection box
previewButton String

The color of the Preview button
previewButtonHover String

The color that is used to emphasize the Preview button when an end user points to it
previewButtonText String

The color of the Preview button text
titlebarBackground

String

The color of the title bar
titlebarText

String

The color of the title bar text
toolbarBackground String

The color of the background of the toolbar
