Get users

The GET API call for users returns a comma-separated list of Empower users who have opened a document in Empower Editor. The list includes the date and time of the most recent sign in for each user, and you can configure it to report activity for up to 365 previous days.

Who can access this call

Access to the Empower API is defined in the customSecurityContext.xml file during installation, and is controlled by your authorization provider. By default, Empower supports the administrator, editor, and integrator authorities.

Authenticated users with the following authority can use this API call:

  • administrator

Request

Request method Syntax
GET /<context>/resource/users?{parameters}

The <context> variable is configured during Empower installation.

The {parameters} search variable is optional. If you do not include any parameters, Empower returns a list of users who have opened a document in Empower Editor within the previous 60 days.

Request parameters

Parameter Data type

Use

 Description
previousDays Integer Optional

Sets the number of days prior to the request to check for user activity.

This parameter can contain a maximum of three digits, with values from 1-365.

Example requests

The following examples show how to create requests using the previousDays parameter. Keep in mind that this API call does not support using a request body.

Example 1:

Request a list of Empower Editor users from the previous 60 days (the default).

Element Example

Request header

GET http://sample-server.example.com:8080/mpw/resource/users

Example 2:

Request a list of Empower Editor users from the previous 24 hours.

Element Example

Request header

 GET http://sample-server.example.com:8080/mpw/resource/users?previousDays=1

Example 3:

Request a list of Empower Editor users from the previous 90 days.

Element Example

Request header

GET http://sample-server.example.com:8080/mpw/resource/users?previousDays=90

Example 4:

Request a list of Empower Editor users from the previous year.

Element Example

Request header

GET http://sample-server.example.com:8080/mpw/resource/users?previousDays=365

Response

This API call returns a comma-separated list of Empower users who have opened a document in Empower Editor, with time stamps.
