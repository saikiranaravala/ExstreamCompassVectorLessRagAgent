Previewing Empower documents

To preview your final document before exporting it for fulfillment, from the editor toolbar, click the Preview button. Depending on your web browser settings, the PDF document opens in a new tab in your browser or downloads to your computer.

Note: Because of variations between web browsers in text wrapping and object placement, the preview document might appear slightly different from the Empower document that you see in your browser.

Ensuring That PDF Preview Documents Open in a New Tab

If PDF preview documents don’t automatically open in a new tab, check your web browser settings to make sure that your local settings support opening links in a new tab. Web sites can specify that a link—such as the PDF Preview button—open in a new tab, but your computer settings can override the web site settings.

If PDF preview documents automatically download but you prefer that they open in your web browser, you might need to add the Adobe PDF add-on or plug-in to your web browser. For additional information about how to display a PDF in your web browser, go to the Adobe Reader web site.
