Understanding how formatting is inherited

When you add a new paragraph to an editable area, the paragraph inherits the following paragraph format settings from the previous paragraph:

  • Left margin
  • Right margin
  • Tab size
  • Indentation
  • Line spacing

Similarly, when you add a new item to an existing bulleted list, the new item inherits the following list item format settings from the previous item:

  • Indent level
  • Bullet character
  • Bullet color

Furthermore, text that you enter in the new paragraph or list item inherits the following character format settings from the last character of the previous paragraph or list item:

  • Font family
  • Font size
  • Bold
  • Underline
  • Italic
  • Strike-through
  • Superscript
  • Subscript
