Identifying editable areas

When theme view is enabled, Empower Editor highlights editable areas, using the following colors:

This color Highlights this type of editable area
  Optional editable areas
  Optional editable areas that have been edited
  Required editable areas
  Required editable areas that have been edited
  Editable areas that contain information that is not formatted correctly (for example, a telephone number field that contains alpha characters)
  Editable areas that are currently disabled or hidden

To toggle theme view, click on the View Menu bar.
