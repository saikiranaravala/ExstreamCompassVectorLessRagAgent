Adding comments to an Empower document

Empower Editor allows you to add comments to the editable text areas of an Empower document, so that you can collaborate with co-workers on the changes that need to be made to a document before sending the final version to customers. Adding comments lets you explain or add context to any changes that you make, or ask questions about changes that other people have made. Additionally, you can take part in discussions within the Empower document itself by adding your comments to existing comments.

Keep in mind that commenting is not available in all Empower documents. If you need to add comments to an Empower document, but the commenting options are not available in Empower Editor, see the person who designed the Empower document.

This section discusses the following topics:

Using the commenting tools

To use the commenting tools in an Empower document, click the Review tab in the Empower Editor menu bar and select from the following options:

Button Action

Show or hide comments in the Empower document.

Add a comment at the current cursor position.

Remove the selected comment thread.

Go to the previous comment in the Empower document.

Go to the next comment in the Empower document.
Adding, editing, or deleting comments

To add, edit, or delete a comment in an Empower document:

  1. Open an Empower document that has the commenting options available.

  2. Do one or more of the following, as needed:

    To Do this

    Add a comment
    1. Within an area of editable text, either select a range of text or place your cursor where you want to add a comment.

    2. On the Review tab, click .

      The comment opens in a separate pop-up window.

    3. Enter the text for the comment.
    4. Click OK.

    Add a comment to an existing comment
    1. Select an existing comment.
    2. On the Review tab, click , or in the comment pop-up window, click .
    3. Enter the text for the comment.

    4. Click OK.

      The new comment is added to the existing comment.

    Edit an existing comment
    1. Select an existing comment.
    2. In the comment pop-up window, click .

    3. Change the text for the comment as needed, then click OK.

      The comment is updated in the pop-up window.

    Delete an existing comment
    1. Select an existing comment.
    2. In the comment pop-up window, click .

    3. Click REMOVE.

      The comment is removed.

    Delete an existing comment thread
    1. Select an existing comment thread.

    2. On the Review tab, click .

      A confirmation message opens.

    3. Click OK.

      The comment thread is removed.

Important: As you work with comments, keep in mind that you can edit or delete only individual comments that you have previously entered. (You can, however, delete comment threads that contain comments from other users.)

Navigating between comments

If your Empower document contains numerous comments, you can use the (previous) and  (next) buttons on the Review tab to navigate between comments, or you can use the following keyboard shortcuts:

Keyboard shortcut Action

SHIFT+ALT+M

Show or hide comments in the Empower document.

CTRL+ALT+M

Add a comment at the current cursor position.

CTRL+ALT+SHIFT+M

Remove the selected comment or comment thread.

CTRL+M

Go to the next comment in the Empower document.

CTRL+SHIFT+M

Go to the previous comment in the Empower document.
ALT + M Enable tab navigation in the currently selected comment window.
TAB Go to the next element in the currently selected comment window (you must first press ALT + M to enable tab navigation).
SHIFT + TAB Go to the previous element in the currently selected comment window (you must first press ALT + M to enable tab navigation).
ENTER Select the highlighted element in the currently selected comment window (you must first press ALT + M to enable tab navigation).
ESC Close the currently selected comment window.
