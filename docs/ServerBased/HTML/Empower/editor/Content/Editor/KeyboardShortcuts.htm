Keyboard shortcuts

The following table details the keyboard shortcuts that are available in Empower Editor:

Functionality Keyboard shortcut Action
General ALT + F1 Open Empower Editor Help.
General ALT + SHIFT + F1 Open a pop-up window with information about which versions of Empower Editor and Empower Services Layer are currently running.
General CTRL + S Save changes to the Empower document.
General CTRL + C Copy the selected content.
General CTRL + X

Cut the selected content.
General CTRL + V Paste copied content from the clipboard.
General CTRL + Z Undo the previous action.
General CTRL + Y Redo the previous action.
Formatting CTRL + B Make the text bold.
Formatting CTRL + I Make the text italic.
Formatting CTRL + U Underline the text.
Formatting CTRL + [ Increase the indent for a paragraph.
Formatting CTRL + ] Decrease the indent for a paragraph.
Formatting CTRL + . Make the text superscript.
Formatting CTRL + , Make the text subscript.
Navigation ALT + SHIFT + P Open the Page Manager.
Navigation CTRL + SHIFT + PAGE UP Go to the first page of the Empower document.
Navigation CTRL + SHIFT + PAGE DOWN Go to the last page of the Empower document.
Navigation SHIFT + PAGE DOWN Go to the next page of the Empower document.
Navigation SHIFT + PAGE UP Go to the previous page of the Empower document.
Comments

SHIFT+ALT+M

Show or hide comments in the Empower document.
Comments

CTRL+ALT+M

Add a comment at the current cursor position.
Comments

CTRL+ALT+SHIFT+M

Remove the selected comment or comment thread.
Comments

CTRL+M

Go to the next comment in the Empower document.
Comments

CTRL+SHIFT+M

Go to the previous comment in the Empower document.
Comments ALT + M Enable tab navigation in the currently selected comment window.
Comments TAB Go to the next element in the currently selected comment window (you must first press ALT + M to enable tab navigation).
Comments SHIFT + TAB Go to the previous element in the currently selected comment window (you must first press ALT + M to enable tab navigation).
Comments ENTER Select the highlighted element in the currently selected comment window (you must first press ALT + M to enable tab navigation).
Comments ESC Close the currently selected comment window.
Editable areas TAB Go to the next editable area in the document.
Editable areas SHIFT + TAB Go to the previous editable area in the document.
Required edit areas CTRL + SHIFT + 8 Go to the previous required edit area in the document.
Required edit areas CTRL + 8 Go to the next required edit area in the document.
Required edit areas CTRL + Q Go to required edit areas that have not changed.

Note: You can use the TAB key on your keyboard to move to the next editable area in your Empower document (and SHIFT + TAB to move to the previous editable area). However, the person who designed the Empower document determines the overall tab order for editable areas on each page of a document. If you have questions about your specific Empower document, see the person who designed the document.
