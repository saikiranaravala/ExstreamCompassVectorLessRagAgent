Pasting images and tables into Empower documents

Empower Editor lets you paste images that are stored in the clipboard and tables that you copy from Microsoft Word into the editable areas of an Empower document.

Note: Because tables that are copied from Word Online have not been tested and might behave differently, the information in this topic pertains only to the client version of Word.

The following list includes a few things to keep in mind when you paste images from the clipboard and tables from the client version of Word:

  • Sometimes, the content that you paste into an Empower document does not keep its original formatting. If you need to restore the original formatting, you can select the affected content in Empower Editor and reapply the formatting as needed.

  • Images and tables cannot be pasted into variable editable areas.

  • Tables should not use the AutoFit Contents setting from Word to automatically resize columns. This setting can cause the columns to adjust to different widths after the table is pasted into your Empower document.

  • In your Empower document, paste tables onto their own line, rather than inline with other text. If you paste a table into a paragraph and the table wraps onto a new line, the Empower preview might not match what you see in Empower Editor.

  • In some designs, paragraphs can be set to overflow, or split, onto the next page if the content becomes too large to fit on the original page. Tables can split if you paste them into a paragraph that is set to split. They cannot split if you paste them into text boxes or other tables. If you have questions about your specific Empower document, see the person who designed the document.

  • If your table contains merged cells and there are merged rows that have different border styles, change the table settings in Word so that the merged rows do not have multiple border styles before you copy and paste the table into your Empower document. Having multiple border styles in the merged rows could cause the height of those rows to be incorrect in your Empower document.

  • Tables that contain only rows with merged cells might appear wider than expected in your Empower document. This issue is cosmetic only and will not affect the final output.
