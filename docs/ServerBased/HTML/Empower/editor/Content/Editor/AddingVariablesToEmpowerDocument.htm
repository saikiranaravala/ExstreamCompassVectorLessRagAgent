Adding variables to an Empower document

Depending on how your Empower document was designed, you might be able to use the Variable Selector to add variables directly to any editable text area. You might want to do this if you are writing a letter that will be sent to multiple customers. In that case, you can add variables to the letter so that it is personalized for each customer who receives the letter. If you have questions about your specific Empower document, see the person who designed the document.

To add variables to an Empower document:

  1. In Empower Editor, on the Menu bar, click the TEXT tab.
  2. Click inside an editable text area at the point where you want to place the variable. (When you place a variable, it will appear at the location of the cursor.)

    3. Note: Variables within the Variable Selector are unavailable when the cursor is inside another variable.

  3. Click to open the Variable Selector.

    The Variable Selector opens in a pop-up window.

  4. From the Variable Selector, double-click the variable that you want to add, or select the variable and click Insert.

    The variable appears in the Empower document.

You can use the Type column in the Variable Selector to differentiate between the types of variables that are available.

Data types represented by variables

The following table lists the different variable types that can be available in the Variable Selector:

Type

Icon

Data referenced

Boolean

Boolean variables represent data that must be one of only two possible values. For example, if the value of a Boolean variable is either true or false, it can have no other values.

Currency

Currency variables represent data that is a monetary amount.

Date

Date variables represent data that is a date, a time, or both.

Floating

Floating variables represent data that is a numeric value that includes decimals.

Formatted text

Formatted text variables represent data that is formatted content that comes from an external repository or file source.

Integer

Integer variables represent data that is a whole number.

String

String variables represent data that contains text.

If the person who designed your Empower document has included an editable array variable (that is, a single variable that contains multiple values), you can use the following keyboard shortcuts to edit the variable:

ENTER—Add a new line to the current array element at the current cursor position

CTRL + SHIFT + A—Add a new element to the array at the current cursor position

Note: If you use this shortcut when your cursor is inside of an existing array element, that element splits into two separate array elements. You can undo that change by pressing BACKSPACE on your keyboard.
