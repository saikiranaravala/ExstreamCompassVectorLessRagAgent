About Exstream Empower Editor

Exstream Empower Editor is a browser-based document interface that lets you edit, preview, and save Empower documents that have been designed and packaged in Exstream Design and Production.

Minimum screen resolution

Empower Editor works best at a screen resolution of at least 1280 x 800.

Supported web browsers

Empower Editor supports the following browsers:

  • Google Chrome, latest stable release
  • Microsoft Edge, latest stable release

  • Microsoft Internet Explorer, version 11 (Enterprise Mode not supported)

  • Mozilla Firefox, latest stable release
Available languages

Empower Editor and the Editor help are available in the following languages:

  • Brazilian Portuguese
  • Chinese
  • Dutch
  • English
  • French
  • German
  • Japanese
  • Spanish

If your web browser is configured to use one of the languages that Empower supports, Empower Editor derives its language settings from your web browser. However, if your web browser is not configured to use one of the supported languages, then Empower Editor uses the default language that the Empower administrator specified for your server.

Supported Input Method Editors

Empower Editor supports the following Input Method Editors (IMEs):

  • Chinese (Simplified): Microsoft Pinyin New Experience Input Style
  • Chinese (Traditional): ChangJie
  • Japanese: Microsoft IME

For information about installing or using different IMEs, see the Microsoft documentation.
