Sending copies of a document to additional recipients

Note: The recipient copies functionality is not currently supported in documents that are previewed or fulfilled through the OT2 platform using Exstream Document Generation on Demand.

Oftentimes, the output that is produced from an Empower document has more than one intended recipient—generally, a primary recipient and then one or more secondary recipients. For example, suppose that you are an insurance agent creating an insurance claim letter in Empower Editor. The claimant (the customer) would receive the letter complete with terms and conditions, and you might need to send the following additional copies of the letter:

  • For an attorney, a copy with the same documents as the claimant
  • For the claimant's agent, a copy without terms and conditions
  • For long-term retention in an archive, a copy with only the legally required information

If the person who designed your Empower document enabled the recipient copies functionality, you can send copies that automatically include the necessary documents for additional recipients.

As with the other functions that are available in Empower Editor, the recipient options available to you are determined by how your Empower document was set up. For instance, you might simply be able to select from a list of additional recipients, or you might also be able to edit the contact information for those recipients. Either way, if recipient functionality is available in your Empower document, you can add new recipients and associate each with a recipient profile that the document designer creates.

This section discusses the following topics:

Selecting which recipients receive a copy of a document

If the recipient copies functionality is available in your Empower document, then you can select who will receive a copy of a document. If you have questions about your specific Empower document, see the person who designed the document.

To select which recipients receive a copy of a document:

  1. In Empower Editor, on the Menu bar, click the VIEW tab.

  2. Click .

    The Select Additional Recipient(s) dialog box opens.

  3. To include or exclude a recipient, select or clear the check box next to each recipient's name, respectively.

  4. Click OK.

    The Select Additional Recipient(s) dialog box closes, and the selected recipients receive a copy of the document after it is finalized.

Adding a new recipient to the recipient list

If the recipient copies functionality is available in your Empower document, then you can add recipients that are not already in the list of available recipients. Each recipient that you add must be associated with a recipient profile, which determines not only the documents that are included with the copy that the recipient receives, but also the contact information that you can specify for the recipient. Recipient profiles are configured by the person who designed your Empower document, and they cannot be changed within Empower Editor. If you have questions about your specific Empower document, see the person who designed the document.

To add a new recipient to the recipient list:

  1. In Empower Editor, on the Menu bar, click the VIEW tab.

  2. Click .

    The Select Additional Recipient(s) dialog box opens.

  3. Click Add Recipient.

    The Add Recipient screen opens.

  4. From the Select a Profile list, select a profile to associate with the recipient that you are adding.

  5. Add recipient information as needed, then click Save.

    The Add Recipient screen closes and the new recipient appears in the list.

Editing recipient information

Depending on how your Empower document was designed, you might be able to edit the information associated with existing recipients. If you have questions about your specific Empower document, see the person who designed the document.

  1. In Empower Editor, on the Menu bar, click the VIEW tab.

  2. Click .

    The Select Additional Recipient(s) dialog box opens.

  3. Hover the pointer over the name of a recipient, and then click .

    The Edit Recipient screen opens.

  4. Update the recipient information as needed, then click Save.

    The Edit Recipient screen closes and the information for the recipient is updated in the Empower document.

Note: The updated information is saved only within your Empower document and does not affect the information contained in the customer database used to create the Empower document.

Previewing an Empower document that includes recipient copies

When you preview an Empower document that includes recipient copies, the contents of each recipient copy in the PDF preview are determined by the settings of the associated recipient profile. Additionally, if the recipient profile uses a cover page, then each recipient copy will be separated by a cover page. If cover pages are not used, then you can use the documents that are contained within each copy to differentiate between recipients. Recipient profiles are configured by the person who designed your Empower document, and they cannot be changed within Empower Editor.

To preview your final document before exporting it for fulfillment, click the Preview button on the Empower Editor toolbar. Keep in mind that because the Empower document might be updated during fulfillment, the preview PDF does not necessarily match the final version of the document that customers and additional recipients will receive. If you have questions about your specific Empower document, see the person who designed the document.

For more information about previewing Empower documents, see Previewing Empower documents.
