Editing and formatting Empower documents

When editing an Empower document, you can edit or format only certain parts, called "editable areas," depending on the design of the document. The document designer might specify that certain text and objects are fixed and cannot be changed, and that some text and objects can be edited or formatted. Furthermore, when formatting text is allowed, the designer might choose to include only certain fonts, or even certain combinations of typefaces, font sizes, font styles, and font weights.

When allowed by the designer, you can format text using the tools on the Text Menu bar.

The text wrapping properties of an editable text area are also set by the document designer. If the text does not wrap as expected or if the text appears to be cut off, the text area has not been designed to support text wrapping.

Note: When you format text, changing a typeface or font size might cause unexpected changes in the font style or weight, depending on the font combinations that the designer included in the document. For example, if Arial Bold is available at a 12-point size, but the bold version is not available at the 10-point size, if you change 12-point Arial Bold text to 10-point, it will no longer be bold.

For more information about editing a specific Empower document, contact the document designer in your organization.
