Starting a Content Author communication

You begin in Communication launcher view in the Exstream web client by selecting from a list of existing communications or by creating a new communication from a list of templates.

To access Communication launcher view, click the Communication launcher button on the application toolbar in the Exstream web client.

If you have questions about the templates that are available, talk to the person or people who designs communications in your Exstream environment.

Creating a new communication

  1. In the Communications panel, click the New button to see a list of available templates.

  2. In the Template name column, select a template. Only templates that have been approved are visible in this column.

  3. If you are not sure which template to select, click Preview to see a preview of a selected template. The Template preview dialog box displays print and HTML previews, as available.
  4. Click Next.

  5. Specify a name, and optionally, a description for the communication that you want to create.

  6. Optionally, in the Category tags applied area, click on the tags that you want to apply. To apply all tags, click Add all.

    You can also use the Search box to search for a specific tag by name.

    Note: This option might not be available in your environment. If you have questions, speak to your Exstream administrator.

  7. Click Create.

    The new communication opens in Communications view in Content Author.

Opening an existing communication

  1. In the Communications panel, click one of the following buttons:

    • Existing—Displays all of the existing communications

    • Recent—Displays a list of communications that you have recently opened in Content Author

    • Favorites—Displays a list of communications that have been bookmarked

      Tip: If you work frequently with a specific communication, you can bookmark that communication so that it appears in the Favorites list. To bookmark a communication, point to the communication in the Library or Recent tabs, and click the Bookmark icon in the Favorites column.

  2. Select the communication that you want to open, and click Open.

    The communication opens in Communications view in Content Author.

Setting measurement units and spelling dictionary language

  1. In Communications view, open or create content.

  2. With the content open in Authoring view, click the hamburger menu , and then click Settings.

  3. In the Settings dialog box, make selections from the following lists as needed:

    • Units of measurement—This is the default unit of measure used when working with content in Authoring view. You can specify inches, centimeters, points, or pixels.

    • Spelling dictionary—This is the default language used for spell-checking in Authoring view. If you have questions about the dictionaries that are available, speak to your system administrator.

  4. Click Save.
