Creating an output object

  1. In the Delivery panel, click the Add button, then click Output in the selection list.

  2. Specify a name, and optionally, a description for the output.

  3. Select an output type.

  4. Click Create.

Updating output object properties

If you want to change the properties of an output object, you can do so by selecting the output object, making the necessary adjustments in the Properties panel, and then clicking Save.

Output object properties vary by output type. For more detail on what properties are available for each type, see Setting up HTML and HTML Email output or Setting up PDF output.

Note: If the output object has been approved, or submitted for approval, the properties cannot be changed.

Duplicating an output object

If you want to create a copy of an output object, you can duplicate it, rather than starting from defaults. This might be useful if you have highly customized the properties of an output object, and need another output object with similar properties but with minor modifications.

To duplicate an output object, in the Communication asset library view, in the Delivery panel, select the output object that you want to copy and click Duplicate.

Deleting an output object

If an output object is not needed, and is not used in any communication objects, it can be deleted from the asset service.

To delete an output object, in the Communication asset library view, in the Delivery panel, select the output object that you want to delete and click the Delete button in the Properties panel.

Tip: If the Delete button is not available, click the Used by button on the References tab in the Properties panel to see if the selected object is used in the current version of a communication object.
