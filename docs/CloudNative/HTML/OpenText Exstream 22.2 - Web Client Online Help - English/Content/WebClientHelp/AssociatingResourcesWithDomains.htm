Associating resources with domains

By default, when you add or create a resource, it is associated with the domain in which it is created. After you approve a resource, you can associate it with other domains as needed, for example, if you created a theme in a development domain and wanted to test it in a test domain. You can view the currently associated domains for a selected resource on the Domains tab in the Properties panel.

You can modify a resource in one domain without affecting other versions of that resource in the other domains that it is associated with. When you associate a resource with a domain, all of its linked resources are also associated with that domain. However, when you remove a domain association from a resource, it does not remove that domain association from its linked resources.

Associating a resource with a domain

Note: You must approve a resource (and all of its linked resources) before you can associate it with a domain.

Exception: You can associate an approved application package with a domain, whether or not its subpackages are approved. Likewise, you can associate an approved subpackage with a domain if at least one version of its associated application package is approved. For more about application package and subpackage resources, see Managing application packages and subpackages.

  1. In the asset library panel, select the resource.
  2. In the Properties panel, click the Domains tab.
  3. In the Available domains list, select a domain, then click the Associate button in that row.
  4. Click Proceed to confirm your action. The associated domain is now displayed in the Currently associated domains list.
Removing a domain association from a resource

Note: You cannot remove the association to the current domain. Additionally, if a resource is only associated with one domain, you cannot remove that domain association from the resource.

  1. In the asset library panel, select the resource.
  2. In the Properties panel, click the Domains tab.
  3. In the Currently associated domain list, select a domain, then click the button in that row.
  4. Click Proceed to confirm your action. The removed domain is now displayed in the Available domains list.
