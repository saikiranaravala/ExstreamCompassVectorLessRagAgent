Creating a custom output format for a variable

If you selected Custom in the Output format list for a variable, you must construct a custom string in the Custom format string box.

Note: This option is only available for string, integer, float, date, and currency output types.

You can construct a custom format string by manually entering characters in the box and by selecting format symbols to add to the string.

To add a custom format symbol to the Custom format string box:

  1. Click the Custom format symbols button .

  2. In the Formatting dialog box, select the custom format that you want to insert.

    If you want to narrow down the list of formatting strings, in the Data type list, select the data type that you want to use to filter the list of available custom formatting options. Currency variables can be modified using the integer data type strings.

  3. Click Insert.

    The Formatting dialog box closes and the format appears in the Custom format string box.

Note:  

  • Additional format symbols are added to the end of the existing string.

  • Some custom formats require you to enter text. For example, if you selected "ABC" replace ABC with your custom text.

Custom format string symbols

You can use the following symbols in the Custom format string box to construct a format string:

Symbol Function
0 Displays a digit or displays zero if there is no digit to display
# Displays a digit or displays nothing if there is no digit to display
. Displays a decimal placeholder
% Multiplies the value by 100 and displays the number as a percentage
, Displays a thousands separator
: Displays a time separator
/ Displays a date separator
E-, E+, e-, or e+ Displays the number in scientific format
- Displays a minus sign
+ Displays a plus sign
$ Displays a dollar sign
( or ) Displays a left or right parenthesis
\ Displays the next literal character
"ABC" Displays the characters within the quotation marks (for example, ABC)
@ Displays a character or displays a space if there is no character to display
& Displays a character or displays nothing if there is no character to display
< Forces all characters to appear in lowercase
> Forces all characters to appear in uppercase
! Forces placeholders to fill left to right
a/p Displays the hour using the 12-hour clock followed by a or p
am/pm Displays the hour using the 12-hour clock followed by am or pm
A/P Displays the hour using the 12-hour clock followed by A or P
AM/PM Displays the hour using the 12-hour clock followed by AM or PM
AMPM Displays AM or PM for the specified hour
c Displays the date and time as m/d/yy h:nn:ss without a leading zero
d Displays the day as a number without a leading zero
dd Displays the day as a number with a leading zero
ddd Displays the abbreviated name of the day of the week
dddd Displays the full name of the day of the week
ddddd Displays the date as m/d/yy without a leading zero
dddddd Displays the date as mmmm d, yyyy (where mmmm is the full name of the month)
DDD Displays the abbreviated name of the day of the week in uppercase
DDDD Displays the full name of the day of the week in uppercase
h Displays the hour as a number without a leading zero
hh Displays the hour as a number with a leading zero
m Displays the month as a number without a leading zero
mm Displays the month as a number with a leading zero
mmm Displays the abbreviated name of the month
mmmm Displays the full name of the month
MMM Displays the abbreviated name of the month in uppercase
MMMM Displays the full name of the month in uppercase
n Displays the minute as a number without a leading zero
nn Displays the minute as a number with a leading zero
q Displays the calendar quarter as a number
s Displays the second as a number without a leading zero
ss Displays the second as a number with a leading zero
ttttt Displays the complete time as h:nn:ss
w Displays the numeric day of the week, 1-7
ww Displays the number of the week within the current year, 0-52
y Displays the day of the year as a number
yy Displays the last two digits of the year
yyyy Displays the year
I

Displays the number of days between January 1, 1900 and the date value stored in the variable
