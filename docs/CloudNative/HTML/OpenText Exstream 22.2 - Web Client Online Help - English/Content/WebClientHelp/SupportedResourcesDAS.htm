Supported DAS resource types

The following table describes the resource types that are supported in DAS and lists the asset library view that you use to manage them.

Supported resource types

Resource type

Description

Asset library view

Barcode

A barcode object that can be used in Communications Designer communications.

Barcodes are available for use only in communications that are based on templates. They are not available to be used in communications that are based on communication sets.

Common panel in Design asset library view

Clause

A design object that is used in Content Author communications

Author panel in Design asset library view

Communication (CA)

A Content Author communication, which is based on an application template that can be created from Design Manager applications or from Communications Designer communications.

Author panel in Design asset library view

Communication (CD)

A Communications Designer communication, which can contain documents and designs such as pages and emails

Designer panel in Design asset library view

Communication set

A collection of other communication assets such as data sources, variables, and output queues.

Communication sets can be used to start new Communications Designer communications with DAS-only resources instead of using a template based on a Design Manager application.

Communications sets panel in Communication asset library view

Component

A design object that can be reused in Communications Designer designs

Designer panel in Design asset library view

Data source

A mapping of variables to a sample data file.

When producing customer output, a data source is used to identify data areas in production data files.

Data panel in Communication asset library view

Design pack

Contains Design and Production resources that can be used in Content Author and Communications Designer. Design packs are created in Design Manager and uploaded to the asset library along with the associated package file.

Design packs panel in Orchestration asset library view

Document

A container object for designs in a Communications Designer communication

Designer panel in Design asset library view

Email

A type of design layout that is used in Communications Designer documents to create designs for HTML Email output

Designer panel in Design asset library view

Flow context

Contains input and output channels and communication settings that are used to set up flow models. Flow contexts are configured in Communications Orchestrator.

Flows panel in Orchestration asset library view

Flow model

Defines the flow for engine orchestration jobs. Flow models contain nodes that receive data from an input channel, process the input data, and deliver the processed data through appropriate output channels. Flow models are configured in Communications Orchestrator.

Flows panel in Orchestration asset library view

Flow script

A resource that lets you create reusable flow scripts in order to implement orchestration logic. Flows panel in Orchestration asset library view

Font

A font object that can be uploaded to DAS and used in Content Author or Communications Designer communications. For communications that are based on Design Manager applications, DAS fonts supplement the fonts that are included in the design pack.

Common panel in Design asset library view

Graphic message

A design object that is used in Content Author communications

Author panel in Design asset library view

Image

Image that can be uploaded to DAS and used in Content Author or Communications Designer

Common panel in Design asset library view

Language

A resource object that defines a language and its settings.

Region panel in Communication asset library view

Locale

A resource object that defines the settings for a geographic location.

Region panel in Communication asset library view

Message type

A design object that is used to insert either a text or graphic message within a message frame in Communications Designer, for use in Content Author communications.

Common panel in Design asset library view

Messaging

A type of design layout that is used in Communications Designer documents to create designs of plain text output for SMS text messages.

 Designer panel in Design asset library view

Metadata

An identification object that is used to sort and classify content in the asset library, and to easily identify resources in output files for use in downstream business processes.

Common panel in Design asset library view

Output

A resource to which you can assign electronic and print output drivers.

Delivery panel in Communication asset library view

Output queue

A resource that contains the settings for output production.

Delivery panel in Communication asset library view

Package

Created when a Design Manager application is packaged and uploaded to DAS from Design Manager. A Package resource is also created for each subpackage that you build and upload to DAS.

Packages panel in Orchestration asset library view

Page

A type of design layout that is used in Communications Designer documents to create designs for page-based output

Designer panel in Design asset library view

Paper type

A layout object that is used to define page characteristics for page designs in Communications Designer communications

Common panel in Design asset library view

Run definition

Contains information that is required to run the Exstream engine and preview output for Content Author and Communications Designer communications. An engine run definition is created either when a Design Manager application that contains a design pack is packaged to DAS or when a communication set is created.

Run definitions panel in Orchestration asset library view

Sample file

A data file that contains sample input data that could be sent to the Exstream engine to produce output. Sample files are used to map data sources and to populate variables in communication previews and simulations.

Common panel in Design asset library view

Section

A design object that is used in Content Author communications

Author panel in Design asset library view

Template

Contains the application content of a Design Manager application.

A template is created when a Design Manager application that contains a design pack is packaged and uploaded to the DAS asset library from Design Manager, or when a Content Author template is created in Communications Designer.

Templates based on Design Manager applications are used to create new Content Author and Communications Designer communications. Templates based on Communications Designer communications are used to create new Content Author communications.

Templates panel in Orchestration asset library view

Text message

A design object that is used in Content Author communications

Author panel in Design asset library view

Variable

DAS variables are created in the data source editor in Mapping view or the Data panel of the Communication asset library panel. They can be used for mapping in a data source and as design objects in a communication.

Data panel in Communication asset library view

Web design

A type of design layout that is used in Communications Designer documents to create designs for HTML5 output

Designer panel in Design asset library view
