Creating metadata

You can use the asset library panel in the Design asset library view to create metadata. The metadata can then be applied to certain resources.

Metadata can be used to sort and classify content in the DAS asset library, and to easily identify resources in output files for use in downstream business processes.

Creating a new metadata tag

  1. In the Common panel in Design asset library view, click the Add button, and then click Metadata in the selection list.

  2. In the Properties panel, provide the following information: 

    • Enter a name, and optionally, a description for the metadata tag.

    • In the Value type list, select one of the following options to specify the type of value that you want to assign to the metadata tag in the output:

      • Use static value—Lets you assign a specific text value to the metadata tag

      • Use variable—Lets you select a variable that will be used to populate the assigned value of the metadata tag in the output. To use a variable from a design pack, you must also select the Use variables from a design pack and then select the design pack in the Design pack list.

    • In the Value box, enter the value that you want to assign to the metadata tag in the output. To specify a variable value, click the Variable button to open a variable selector panel, and then select the appropriate variable.

    • To include the metadata in the output manifest, select the Include in output manifest check box.
