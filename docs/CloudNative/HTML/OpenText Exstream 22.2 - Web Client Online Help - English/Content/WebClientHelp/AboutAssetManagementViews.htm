Asset library views at a glance

The asset library views in the Exstream web client let you add and manage resources that are used to create communications and generate customer output. Depending on the type of resource that you want manage, you use one of following asset library views.

Communication asset library view

You use the Communication asset library view to add and manage the resources that are used to configure Exstream communications. To access this view, click the Communication asset library button on the application toolbar in the Exstream web client.

The Communication asset library view contains the following panels:

  • Communication sets—Lets you combine other communication assets into one collection

  • Data—Lets you create and manage variables and data sources for use in communications

  • Delivery—Lets you create and manage output drivers and queues

  • Region—Lets you create and manage languages and locales

When you select a resource, a corresponding Properties panel lets you view and modify the properties for that selected resource. The Properties panel is a dynamic panel that is always located on the right side of the view.

For information about adding resources in this view, see Adding communication resources.

Design asset library view

You use the Design asset library view to add and manage the design resources that are used to create and design Communications Designer and Content Author communications. To access this view, click the Design asset library button on the application toolbar in the Exstream web client.

The Design asset library view contains the following panels:

  • Common—Lets you add and manage design resources that are not specific to Communications Designer or Content Author
  • Designer—Lets you manage resources that are related to Communications Designer communications
  • Author—Lets you manage resources that are related to Content Author communications

When you select a resource, a corresponding Properties panel lets you view and modify the properties for that selected resource. The Properties panel is a dynamic panel that is always located on the right side of the view.

If a Communications Designer or Content Author resource uses variations, you will see the number of alternate variations associated with the resource in the Variations column. Click the number of variations in the Variations column to expand the list of alternate variations.

Important: When the standard variation of a resource is in draft state, users can delete any version of an approved alternate variation from Design view in Communications Designer and from Authoring view in Content Author. In Design asset library view, you do not have the option to delete only alternate variations. When you delete a standard variation in Design asset library view, all alternate variations are also deleted.

For information about creating variations in Communications Designer and Content Author, see OpenText Exstream Communications Designer Online Help and OpenText Exstream Content Author Online Help, respectively.

For information about adding resources in this view, see Adding design resources.

Orchestration asset library view

You use Orchestration asset library view to add and manage the resources that are used preview communications and create customer output using the Exstream engine. To access this view, click the Orchestration asset library button on the application toolbar in the Exstream web client.

Note: If you are using the OT2 platform, Orchestration asset library view is only available if the Exstream subscription is set to Standard (not Base).

The Orchestration asset library view contains the following panels:

  • Flow—Lets you add and manage flow models and flow contexts for engine orchestration jobs that are defined in Communications Orchestrator

  • Templates—Lets you manage application templates that are associated with Design Manager applications and Communications Designer communications

  • Design packs—Lets you manage design packs that are uploaded along with Design Manager application packages

  • Packages—Lets you manage application packages and subpackages that were created in Design Manager and uploaded to DAS from Design Manager.

    In the Packages panel, the Subpackages column displays the number of subpackages, if any, that are associated with the application package. Click the number of subpackages to expand the nested list of subpackages.

    For information about creating subpackages and uploading them to DAS, see OpenText Exstream Design and Production Online Help.

    For rules that apply to application package and subpackage resources, see Managing application packages and subpackages.

  • Run definitions—Lets you manage run definitions for engine orchestration jobs that uploaded along with Design Manager application packages

When you select a resource, a corresponding Properties panel lets you view and modify the properties for that selected resource. The Properties panel is a dynamic panel that is always located on the right side of the view.

For information about adding resources in this view, see Adding orchestration flow resources.

Monitor view

You use Monitor view to see the flow instances that correspond to your engine orchestration jobs and the status of each job. To access this view, click the Monitoring button on the application toolbar in the Exstream web client.

Note: If you are using the OT2 platform, Monitor view is only available if the Exstream subscription is set to Standard (not Base).

The Monitor view contains the following panels:

  • Jobs—Lets you manage engine orchestration jobs

  • Flows—Lets you manage the flow instances associated with a specific orchestration job

When you select a resource, a corresponding Properties panel lets you view the properties for that selected resource. The Properties panel is a dynamic panel on the right side of the view.

For information about monitoring orchestration jobs, see Monitoring orchestration jobs.

Rationalization asset library view

You use Rationalization asset library view to add and manage the migration sets that are used in Communications Designer communications. To access this view, click the Rationalization asset library button on the application toolbar in the Exstream web client.

Note: Licensing for Exstream CE Rationalization requires a services engagement with OpenText Professional Services. For more information, contact your Exstream account representative.

The Rationalization asset library view contains the Migration sets panel. When you select a resource in the Migration sets panel, a corresponding Properties panel lets you view and modify the properties for that selected resource. The Properties panel is a dynamic panel that is always located on the right side of the view.

For specific information about adding resources in this view, see Rationalizing external content.
