Creating configurations for Communications Designer

You can use the Configurations panel in Tenant management view to make the following configurations for Communications Designer:

On the General tab:

Effectivity

You can select the Use effectivity check box if you want to let designers apply effective dates to documents, designs, and components in Communications Designer. You can also set effective and expiration dates on documents, designs, and components in Design asset library view; however, designers will be able to change that setting later on in Communications Designer.

Experience AI Services

You can select the Use Experience AI Services check box to give designers in Communications Designer access to a suite of Assisted Authoring tools in Communications Designer that provide text analysis to measure readability, sentiment and tone, and grammar. To enable this functionality in Communications Designer, you must specify a URL to connect to Experience AI Services. You can use the Run validation button just below the Connection URL box to check that the Experience AI Services URL is properly connecting to the service.

To make the Assisted Authoring tools available to your deployment, you must also separately deploy Experience AI Services into your Kubernetes cluster. For information about deploying and configuring Experience AI Services, see OpenText Experience AI Services Configuration Guide, available on My Support.

For information about using Assisted Authoring tools in Communications Designer, see OpenText Exstream Communications Designer Online Help.

On the Text and Fonts tab:

For communications based on a communication set, you can set the default font to be used when creating designs in Communications Designer. You can also set the default bullet and numbering formats used for up to four levels of indentation. These defaults will apply to all supported designs created in the tenant.

On the Dictionaries tab:

You can use the settings on the Dictionaries tab to manage installed spelling dictionaries for Communications Designer.
