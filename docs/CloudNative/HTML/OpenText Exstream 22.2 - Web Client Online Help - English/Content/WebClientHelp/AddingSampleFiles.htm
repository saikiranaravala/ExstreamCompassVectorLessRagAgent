Adding sample files

You can use the asset library panel in the Design asset library view to make the following sample file types available for use with communications and templates:

  • Driver

  • Initialization

  • Reference

  • Report

  • Auxiliary layout

  • Empower driver

Adding a sample file

  1. In the Common panel, click the Add button, then click Sample file in the selection list.

  2. In the file browser, select the sample file that you want to add, then click Open.

  3. Optionally, select the newly added file in the Common panel and on the General tab in the Properties panel, enter a description to help distinguish it from other sample files and select the appropriate File type.
Updating an existing sample file

If a sample file is in draft state, you can replace the existing resource in the asset library with another file. Replacing a resource does not create a new version of the existing resource. To use the updated version of a sample file in a communication preview, you must approve the sample file resource.

To replace a sample file:

  1. In the Common panel, select the sample file.
  2. In the Properties panel, on the General tab, click the Replace button .
  3. In the file browser, select the file that you want to upload to replace the existing sample file, then click Open.
  4. Optionally, update the name and description of the sample file.
  5. Click Save.
