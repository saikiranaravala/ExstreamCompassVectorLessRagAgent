Creating paper types

You can use the asset library panel in the Design asset library view to create paper types for use with your communications. A paper type resource is used to define the size, and color of paper stock. Paper types can also define the screen properties for electronic output.

At least one paper type is required to create a page design in Communications Designer. Designers can select paper types that are created in Design asset library view as well as paper types that are defined in Design Manager applications.

To create a new paper type:

  1. In the Common panel, click the Add button, then click Paper type in the selection list.

  2. In the Properties panel, provide the following information:

    • Enter a name, and optionally, a description for the paper type resource.

    • To define the dimensions of the paper type, enter the appropriate values in the Width box and Height box. To change the units of measurement, go to > Settings, and select the desired unit in the Units of measurement list.

    • To specify a background color for the paper type, click the Paper color well and select a color from the color palette, then click Apply.

  3. Click Create.

    The paper type resource is created with the specified settings and the Properties panel displays the paper type properties. You can use the Properties panel to update the properties as needed.
