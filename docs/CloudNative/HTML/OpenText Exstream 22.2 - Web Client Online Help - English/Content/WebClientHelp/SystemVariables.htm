Using system variables

System variables let you use system information, such as the information stored in the design environment, to complete tasks such as monitoring and controlling an engine run, formatting designs, or organizing a customer mailing.

While some DAS system variable properties are predefined, you can edit the Design sample and Output format fields in the Properties panel. You cannot delete a system variable.

System variables can be categorized as follows:

Note: In the tables below, if a system variable name is in bold, its value can be calculated using a rule.

Communication design related system variables

These variables are related to the objects required to create a communication: communications, documents, and designs.

Communication design related system variables

Variable name

Description

SYS_DocumentNames

The names of the documents included in the customer.

SYS_PageInDocument

The sequence number of the current counted page within the document.

The value of the 'SYS_PageInDocument' system variable might vary from the value of the 'SYS_SheetInDocument' system variable if duplex pages are included in a document.

SYS_PagePhysicalInDocument

The sequence number of the current physical page within the document.

The number of physical pages might differ from counted pages because of the page counting methods set in the communication.

SYS_PagePrintedValue

The value to use as the page number of the current page.

The page number might differ from page count because the page count might have been restarted or because of blank pages.

SYS_PageStart

The initial number of pages included in the document before pages are excluded for reasons such as rules or marketing page limits.

SYS_PageTotalInDocument

The total number of counted pages included in the document.

The value of the 'SYS_PageTotalInDocument' system variable might vary from the value of the 'SYS_SheetTotalInDocument' system variable when duplex pages are included in a document.

SYS_PageTotalPhysicalInDocument

The total number of physical pages included in the document.

The number of physical pages might differ from counted pages because of the page counting methods set in the communication.

SYS_PageTotalPrinted

The page number of the last page in the document.

The value of the 'SYS_PageTotalPrintedInDocument' system variable might vary from the value of the 'SYS_PageTotalPhysicalInDocument' system variable when page counts reset at the start of a document.

SYS_SheetInDocument

The sequence number of the current sheet within the document.

The value of the 'SYS_SheetInDocument' system variable might vary from the value of the 'SYS_PageInDocument' system variable when duplex pages are included in a document.

SYS_SheetTotalInDocument

The total number of sheets included in the document.

The value of the 'SYS_SheetTotalInDocument' system variable might vary from the value of the 'SYS_PageTotalInDocument' system variable when duplex pages are included in a document.

SYS_SubDocInDocument

The sequence number of the current sub-document (design document ) within the current document (customer).

SYS_TotalSubDocsInDocument

The total number of sub-documents (design documents) in the current document (customer).
Counting and numbering related system variables

These variables are related to counting a type of object in a communication or numbering an object as it is processed.

Counting and numbering related system variables

Variable name

Description

SYS_CustomerInRun

The sequence number of the current customer within the engine run.

SYS_DocumentTotalInRun

The total number of documents included in the engine run.

SYS_PageInBreak

The sequence number of the current page within the convenience break of the output queue.

SYS_PageInDocument

The sequence number of the current counted page within the document.

The value of the 'SYS_PageInDocument' system variable might vary from the value of the 'SYS_SheetInDocument' system variable when duplex pages are included in a document.

SYS_PageInQueue

The total number of physical pages included in the current output queue.

SYS_PagePhysicalInDocument

The sequence number of the current physical page within the document.

The number of physical pages might differ from counted pages because of the page counting methods set in the communication.

SYS_PagePrintedValue

The value to use as the page number of the current page.

The page number might differ from page count because the page count might have been restarted or because of blank pages.

SYS_PageStart

The initial number of pages included in the document before pages are excluded for reasons such as rules or marketing page limits.

SYS_PageTotalInDocument

The total number of counted pages included in the document.

The value of the 'SYS_PageTotalInDocument' system variable might vary from the value of the 'SYS_SheetTotalInDocument' system variable when duplex pages are included in a document.

SYS_PageTotalInRun

The total number of pages included in the run.

SYS_PageTotalPhysicalInDocument

The total number of physical pages included in the document.

The number of physical pages might differ from counted pages because of the page counting methods set in the communication.

SYS_PageTotalPrinted

The page number of the last page in the document.

The value of the 'SYS_PageTotalPrinted' system variable might vary from the value of the 'SYS_PageTotalPhysicalInDocument' system variable when page counts reset at the start of a document.

SYS_SheetInBreak

The sequence number of the current sheet within the convenience break of the output queue.

SYS_SheetInDocument

The sequence number of the current sheet within the document.

The value of the 'SYS_SheetInDocument' system variable might vary from the value of the 'SYS_PageInDocument' system variable when duplex pages are included in a document.

SYS_SheetInQueue

The total number of sheets created using the output queue.

SYS_SheetTotalInDocument

The total number of sheets included in the document.

The value of the 'SYS_SheetTotalInDocument' system variable might vary from the value of the 'SYS_PageTotalInDocument' system variable when duplex pages are included in a document.

SYS_TableRow

The sequence number of the current automated table row.

If the row repeats, the value of the 'SYS_TableRow' system variable indicates the current count of the row, including those rows that were not composed.

Note: You can use the 'SYS_TableRow' system variable in formulas to compute transaction data on a row-by-row basis. The 'SYS_TableRow' system variable does not work for embedded tables.

SYS_TotalSubDocsInDocument

The total number of sub-documents (design documents) in the current document (customer).
Customer related system variables

These variables are related to information about a customer.

Customer related system variables

Variable name

Description

SYS_CustomerInRun

The sequence number of the current customer within the engine run.

SYS_DocumentNames

The names of the documents included in the customer.

SYS_LanguageCustomer

The language of the current customer, which is used to identify the values to use when dynamically selecting language-based objects and settings.

SYS_LanguageDefault

The default language for all of the customers within the engine run.

SYS_LocaleCustomer

The locale for the current customer, which is used to identify the variables that dynamically select the appropriate locale-based objects and settings.

SYS_LocaleDefault

The default locale for all of the customers within the engine run.

SYS_TotalSubDocsInDocument

The total number of sub-documents (design documents) in the current document (customer).
