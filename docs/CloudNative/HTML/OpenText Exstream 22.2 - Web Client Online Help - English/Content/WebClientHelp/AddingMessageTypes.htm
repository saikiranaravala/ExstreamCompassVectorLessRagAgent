Creating message types

In Communications Designer, you can add content frames and messages frames to indicate content areas in a design that are available for editing in Content Author.

  • Content frames let users add flowing paragraph content to customer communications, and are the only type of frames that you can use for a Content Author communication that will be edited in Empower Editor. For more information about adding content frames in Communications Designer communications, see OpenText Exstream Communications Designer Online Help.

  • Message frames let users add text and images in a content area in Content Author. Message frames can contain the following types of messages:

    • Text message contain content that is placed inline in the frame. Text messages can grow to accommodate content and can split and flow to the next text message frame. Text messages are best suited for text-heavy content that is limited in scope, such as sidebars or callout boxes.
    • Graphic messages contain text and image content in a content area of pre-defined dimensions. The size of a graphic message frame is static. Because of this, graphic messages are best suited for image-heavy pieces of content that need to fit within a specific area and that do not need to flow.

The size and color of message frames is determined by message types, and at least one message type is required to insert a message frame in Communications Designer. Message frames can use message types that are created in DAS as well as message types and message templates that are defined in Design Manager applications. In DAS, you use the asset library panel in Design asset library view to create DAS message types. For information about creating message types in Design Manager, see OpenText Exstream Design and Production Online Help.

To create a new message type in DAS:

  1. In the Common panel, click the Add button, then click Message type in the selection list.

  2. In the Properties panel, provide the following information:

    • Enter a name, and optionally, a description for the message type resource.

    • In the Type list, select Text or Graphic as the message type.

  3. For Graphic message types, specify the following additional settings in the Properties panel:

    • To define the dimensions of the graphic messages that are based on this message type, enter the appropriate values in the Width box and Height box. To change the units of measurement, go to > Settings, and select the desired unit in the Units of measurement list.

    • To specify a background color for graphic messages that are based on this message type, click the Color well and select a color from the color palette, then click Apply.

  4. Click Create.

    The message type resource is created with the specified settings, and the Properties panel displays the message type properties. You can use the Properties panel to update the properties as needed.
