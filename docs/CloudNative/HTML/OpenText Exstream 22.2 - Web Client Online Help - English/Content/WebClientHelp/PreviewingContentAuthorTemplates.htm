Previewing Content Author templates

When you preview a Content Author template or communication, the preview displays print and HTML previews, as available. You can preview Content Author communications in Communications view and in Simulation view. For information about these communication previews and simulations, see OpenText Exstream Content Author Online Help.

In the Exstream web client, you can preview Content Author templates in Communication launcher view and in Orchestration asset library view.

  • In Communication launcher view, you can preview the approved templates in the list of available templates. This functionality users preview and select a template to create a new communication.
  • In Orchestration asset library view, you can preview a template regardless of its workflow state. This functionality is especially useful when you update a template because you can preview the draft template before you approve it and make it available to Content Author users.

For previews to work correctly, the engine run definition associated with the template must reference an engine version that is present in your deployment.

Previewing a template in Communication launcher view

  1. In the Communications panel, click the New button to see a list of available templates.

  2. In the Template name column, select a template. Only approved templates are visible in this column.

  3. Click Preview to see the template preview.
  4. In the Template preview dialog box, select the output preview type as needed.
Previewing a template in Orchestration asset library view
  1. In the Templates panel, select a template.
  2. On the General tab in the Properties panel, click the Preview button next to the template name.

  3. In the Template preview dialog box, select the output preview type as needed.

    Note: If the selected template does not contain any Content Author frames, preview output is not displayed.
