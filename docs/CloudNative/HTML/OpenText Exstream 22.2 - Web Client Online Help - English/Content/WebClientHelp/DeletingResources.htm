Deleting resources

If a resource is not needed, it can be deleted from the asset library. Deleting a resource deletes it from only the current domain.

When you delete a resource, you can choose to either delete the current version or to delete all versions of the resource from the asset library. You can delete a specific version of a resource only if the selected resource has multiple versions. If the selected version is used by other resources, the workflow states of the selected resource and any of its parent resources determine whether you can delete the resource. Use the References tab in the Properties panel to see which resources use the selected resource, and keep the following considerations in mind:

  • To delete a resource that is used by multiple parent resources that are all in draft state, at least one other version of the resource must exist in the current domain.
  • To delete a resource that is used by any approved parent resource, at least one approved version of the selected resource must exist in the current domain.

Important:  

  • Before you delete a flow model or flow context, make sure that you first unpublish the version that you want to delete. Deleting a published flow resource does not unpublish it. Special considerations apply when deleting published flow models or flow contexts. For more information, see Considerations for deleting flow resources.

  • You can delete an application package only after you delete its subpackages. For more about application package and subpackage resources, see Managing application packages and subpackages.

Deleting a specific version of a resource
  1. Select the resource in the asset library panel.

  2. In the Properties panel, click the version selector, and then select the version that you want to delete. The version selector is available only if more than one version of the selected resource exists.

  3. Click the Delete button in the Properties panel. The confirmation dialog box specifies the version of the resource that will be deleted.

  4. Click Delete to confirm your action. The selected version is deleted.
Deleting all versions of a resource
  1. Select the resource in the asset library panel.

  2. Click the Delete button in the Properties panel. The confirmation dialog box specifies the version of the resource that will be deleted.

  3. To delete all versions of the resource, select the Delete all versions check box.

    Note: The Delete all versions check box is available only if the selected resource is not used by any other resources. You can use the References tab in the Properties panel to see the resources that use a selected resource.

  4. Click Delete to confirm your action. The selected version is deleted.
