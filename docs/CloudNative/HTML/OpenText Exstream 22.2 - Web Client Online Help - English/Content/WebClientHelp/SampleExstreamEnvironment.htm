Sample tenant environment

Exstream deployment parameters include the option to configure a sample tenant that contains default domain and user settings that you can use to familiarize yourself with the deployment.

The sample tenant is created automatically during deployment only if automatic schema management is enabled for Exstream database services. If you are using manual schema management for your Exstream databases, then you must separately create the sample tenant schemas and then either provide the schema information as bootstrap configuration parameters during initial deployment, or provide the sample bootstrap configuration parameters in a helm upgrade command after initial deployment. For more information about using bootstrap configurations during deployment, see OpenText Exstream Installation and Configuration Guide.

Note: You cannot delete the sample tenant in your Exstream environment.

To use the sample tenant, you will need a temporary product key. Contact your Exstream account representative to obtain this key. You can access this tenant at the following link:

http://<host>:<optionalPort>/design/index.html?tenant=sample

To use this tenant for the first time, use the default tenant administrator user name tenantadmin and password Exstream1! to authenticate your connection.
