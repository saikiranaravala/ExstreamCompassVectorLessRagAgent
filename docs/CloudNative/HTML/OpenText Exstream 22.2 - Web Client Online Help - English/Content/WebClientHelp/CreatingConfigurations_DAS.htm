Creating configurations for Design asset library view

You can use the Configurations panel in Tenant management view to specify tenant-wide defaults for design assets in Exstream.

Media Management Services

Select the Use Media Management Services check box and then specify the Media Management URL to connect to the service, and the username and password to connect to Media Management during fulfillment. You can optionally specify the name of a Media Management AMD profile from those that currently exist in the Media Management service. The automatic changes to the image prescribed by the profile are applied when the image is inserted in Design view. You can use the Run validation button to verify that Exstream is properly connecting to the service.

Before they can access images stored in Media Management for their designs, Communications Designer users must log in to Media Management in Exstream. They will be prompted to do so when they insert an image in Design view.

For more information about deploying Media Management, see OpenText Media Management Cloud Deployment Guide. For information about specific configurations required to integrate Media Management with your Exstream deployment, see OpenText Exstream Installation Guide. For information about managing permissions in OTDS, see OpenText Directory Services Installation and Configuration Guide.
