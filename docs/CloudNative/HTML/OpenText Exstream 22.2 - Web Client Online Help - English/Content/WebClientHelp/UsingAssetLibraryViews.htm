Viewing resources in asset library views

In the asset library views, you perform the following general tasks to customize the displayed resource lists in the asset library panel and to view common resource properties.

Note: When resources are created or updated in DAS, the time-stamp is recorded in Coordinated Universal Time (UTC). Therefore, filtering or sorting by date may return unexpected results.

Filtering displayed resources in the asset library panel

By default, asset library panels display all of the available resources for each panel. To make it easier to locate a specific resource, you can filter the list of displayed resources. The following table describes the different methods you can use to filter the displayed list.

Resource list filters

Filter type

Description

Resource name

To filter the displayed list by resource name, use the Search box in the top right corner of the asset library panel.

Tip: You can use the Search box as follows to help you find items in the list:

  • Enter a full or partial search term in the box and then click the search button .

  • Click the arrow in the box to show the Advanced search options, select one or more filters, and then click Search. For more information, see Filtering searches.

In the Design asset library view only: If your deployment supports the advanced search option Search text within content, you can find resources in the list that contain messages and clauses with text that matches the search term.

Resource type

To filter the displayed list by resource type, click the resource type selection list and select from the available options. You can select as many resource types as you want. To select all types, simply clear all existing selections.

Resource properties

  1. Click the Filter button , specify the following settings as needed:

    • In the Created area, use the From and To boxes to enter a range for the resource creation date. You can also click the calendar button to select a date instead of manually entering it.
    • In the Updated area, use the From and To boxes to enter a range for the date that the resource was last updated. You can also click the calendar button to select a date instead of manually entering it.

    • In the Workflow list, select a resource workflow state.

      Note: When you filter by Approved, you see the latest approved version of an object, even if there is a later version in a non-approved workflow state.

  2. Click Apply.

Note: Because of the relationship between standard variations and any associated alternate variations, filters are ignored for variations. If your filter includes a standard variation, then all of the alternate variations will appear in the sub-list beneath the standard variation—even if your filter should exclude one or more of the alternate variations.

By default, the first 25 items are displayed in the asset library panel. To change the number of items displayed in the panel, use the Items per page list. If the number of resources is too large to display in the panel, you can navigate the list with the page buttons .

Bookmarking a resource

If you work frequently with a specific resource, you can bookmark that resource so that it appears in the Favorites list in the asset library panel. To bookmark a resource, point to the resource in the asset library panel, and then click the Favorite icon in the Favorites column.

To see the list of bookmarked resources in an asset library panel, click the Favorites button. To return to viewing all resources, click the Library button.

Customizing the columns in the asset library panel

By default, asset library panels show the following information for each resource: Name, Type, Version, Created, Updated, Favorites.

If you want to customize the information that is displayed for each resource, use the Column selections list to select or clear applicable check boxes.

Hiding resources in the asset library

To hide a selected resource, in the Properties panel, click the Visible resource button.

When you hide a resource, all versions of that resource are hidden in all of the views in the Exstream web client. However, you can view hidden resources in the asset library by clicking the Show hidden assets button in the asset library panel. To go back to viewing only visible resources, click the Show hidden assets button again.

Note: If a resource is in use and then subsequently hidden, it will still be visible where used. The reference is retained to preserve the current design functionality, but the resource will not be available to be used in a new instance.

To unhide a previously hidden resource, select the resource in the asset library panel, and then in the Properties panel, click the Hidden resource button.

Viewing advanced resource details

You can select a resource and see additional information about it that is stored in the asset library.

To see these advanced details for a selected resource, click the Advanced button on the General tab in the Properties panel. The following table describes the information that is displayed for the selected resource.

Advanced details

Item

Description

Name

The name of the resource

ID

The DAS resource identifier string for the resource

Version

The current version of the resource. The version number is incremented every time that a new draft is created from an approved version of the resource.

Content type

The type of resource

Description

The description of the resource, if available

Created

The user that created the resource, and the date and time that it was created

Updated

The user that last updated the resource, and the date and time that it was updated

Reserved by

The user, if any, that has locked the resource. This information is available only if the selected resource is locked for editing.

Effective date

The start date on which the resource is available to be included in customer output. This information is available only for Content Author objects.

Expiration date

The date on which the resource is no longer available to be included in customer output. This information is available only for Content Author objects.

Design pack

The name, version, and DAS resource ID of the design pack that is associated with the communication. This information is available only for Communications Designer communications.

Communication set

The name, version, and DAS resource ID of the communication set that is associated with the communication. This information is available only for Communications Designer communications.

Data source

The name, version, and DAS resource ID of the data source that is associated with the communication. This information is available only if for Communications Designer communications that use a DAS data source.
Viewing resource workflow history

To see the workflow history for a selected resource, in the Properties panel, click the Workflow tab. On this tab, you can see the version history for the resource, the workflow events related to each version, and any comments that were added for each workflow state change.

Viewing resource references

You can use the References tab in the Properties panel to see all of the linked resources for a selected resource, if applicable.

To see the references for a resource:

  1. In the asset library panel, select a resource.

  2. In the Properties panel, click the References tab.

  3. To display a list of references for the selected resource, click the following buttons:

    • Used by—The list shows all of the parent resources that use the selected resource. Keep in mind that this list displays only parent resources with a direct link to the selected resource. Resources that use the parent resources are not listed. This option is selected by default.

    • Uses—The list shows all of the child resources that are used by the selected resource, including resources that are used by each child resource.

  4. To filter the displayed list by resource type, click the resource type selection list and select from the available options. You can select as many resource types as you want. To select all types, simply clear all existing selections.

By default, the first 50 items are displayed in the Properties panel. To change the number of items displayed in the panel, use the Items per page list. If the number of resources is too large to display in the panel, you can navigate the resource list with the page navigation buttons .
