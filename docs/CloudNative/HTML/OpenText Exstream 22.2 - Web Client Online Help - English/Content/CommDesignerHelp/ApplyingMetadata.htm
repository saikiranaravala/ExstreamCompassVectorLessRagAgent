Applying metadata tags

Metadata provides the ability to easily identify resources in output files for use in downstream business processes, such as with Core Experience Insights, and in output post-processing, such as applying additional CSS styles to specific design objects.

Metadata tags are stored as DAS resources and can be managed in the same way as other DAS resource types. For information about creating metadata tags and managing resources in DAS, see OpenText Exstream Web Client Online Help.

In Structure view, you can apply metadata tags to the documents, designs, and components in your communication, as well as to the communication itself.

In Design view, you can apply metadata tags to individual objects in your design.

Applying metadata to communication objects

  1. With the object selected, click the Metadata tab in the Properties panel.

  2. Click the Assign metadata button .

  3. In the Metadata dialog box, select the check box for each metadata tag that you want to apply to the resource.

    Note: The metadata tags that are available in this dialog box are created in Design asset library view in the web client. For more information about creating metadata tags, see OpenText Exstream Web Client Online Help.

  4. Click OK to close the dialog box, and then click Save.

Applying metadata tags to design objects

To apply metadata to most design objects, you use the object properties card in the properties panel in Design view, with the following exceptions: 

  • For barcodes, you must apply metadata tags from Design asset library view in the web client.
  • For components, you must apply metadata tags in Structure view or in Design asset library view in the web client.

To apply metadata tags in Design view:

  1. In the object properties card in the properties panel, click the Metadata button .

    Note: You cannot apply metadata to frames or to tables of contents.

  2. Click Add.

  3. In the Metadata dialog box, select the check box for each metadata tag that you want to apply to the resource.

    Note: The metadata tags that are available in this dialog box are created in Design asset library view in the web client. For more information about creating metadata tags, see OpenText Exstream Web Client Online Help.

  4. Click OK.
Removing applied metadata

To remove an applied metadata tag, click the Remove metadata button for that tag and then save your changes.
