Restricted switches for containerized Exstream deployments

The following switches cannot be used for on-demand or batch Exstream engine runs in the containerized Exstream deployment:

  • ADDINDIRECTORY
  • ALL_ADVANCED_CHARTS_JAVASCRIPT_EXTERNAL
  • ALLINMEM
  • APP_PACKAGEFILES
  • CAPTURETESTDATA
  • CERTIFICATE_FILE_PATH
  • CLOSE_OUTPUT_ON_BREAK
  • CONNECTORMAP
  • CONTROLDDA
  • CONTROLFILE
  • CONTROLFILEENCODING
  • COPYINPUT
  • CREATE_RESMANAGERFILE
  • CREATE_REUSABLE_RESMANAGERFILE
  • CREATEOUTPUTSUBDIRECTORIES
  • CUSTOMERLISTFILE
  • DDAFILEMAP
  • DDAMESSAGE
  • DDAOUTPUT
  • DDAOUTPUTFILE
  • DEVICEPREVIEW
  • DISABLEPRINT
  • DOCGENRUN
  • DOCX_COLLECT_GARBAGE_EACH_CUSTOMER
  • DOCX2DXF_LOCATION
  • DYNAMSG or DYNAMESSAGE
  • EXPOIWRAPPER_CLASSPATH
  • EXSTREAM_DTD_LOCATION
  • GS_PAUSE
  • GSTEMPDIR
  • HTML_DTD_LOCATION
  • IMPORTDIRECTORY
  • KEY
  • KEYFILE
  • KEYPART
  • LICENSE_PATH
  • LICENSE_WAIT
  • LOADREFFILE
  • LOGDDA
  • MESSAGEFILE
  • MSGRESOURCE
  • ON_PREMISE_TOKEN
  • ONDEMAND
  • OUTPUTDIRECTORY
  • OUTPUTFILE
  • PACKAGED_FILEMAP
  • PACKAGEFILE
  • REALTIME
  • REALTIME_LICENSE_MODE
  • REPORTFILE
  • RESMANAGERFILE
  • REUSABLE_RESMANAGERFILE
  • RULEANALYSISREPORT
  • RUNSUMMARY
  • SMARTPROOF
  • TRACEFILE
  • TRACK_OVERRIDE_VERSION_CHECK

  • TRACKOUT
