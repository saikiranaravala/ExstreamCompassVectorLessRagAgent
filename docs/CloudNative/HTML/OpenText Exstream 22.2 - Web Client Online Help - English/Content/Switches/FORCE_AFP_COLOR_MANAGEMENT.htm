FORCE_AFP_COLOR_MANAGEMENT

Use the FORCE_AFP_COLOR_MANAGEMENT engine switch to use the AFP color management architecture functionality for images when you produce AFP output from an application that imports images at run time and your application includes the following settings:

  • The Place at top of print stream option is selected from the Image processing drop-down list in the Image management area on the Resource Management tab of the AFP output object properties.
  • The Used resources only option is selected from the Resource inclusion drop-down list in the File resource management area on the Resource Management tab of the AFP output object properties, or you have included a resource management file specified by the RESMANAGERFILE engine switch in your control file.

As an alternative to this switch, you can select the AFP color management architecture check box on the Basic tab of the AFP output object properties.

Syntax

-FORCE_AFP_COLOR_MANAGEMENT=<option>
Arguments Required Supported values
option Yes
  • ON—Specifies that the engine uses AFP Color Management architecture when it processes images for AFP output
  • OFF—Specifies that the engine does not use AFP Color Management architecture when it processes images for AFP output

Example

-FORCE_AFP_COLOR_MANAGEMENT=ON

Supported in

7.0.303 and later versions
