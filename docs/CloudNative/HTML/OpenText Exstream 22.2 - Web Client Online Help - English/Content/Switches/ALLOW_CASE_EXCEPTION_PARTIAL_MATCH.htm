ALLOW_CASE_EXCEPTION_PARTIAL_MATCH

Use the ALLOW_CASE_EXCEPTION_PARTIAL_MATCH engine switch when you run an application that references a case exception file that contains a case exception for phrases. In Exstream, the case exception file must be verbatim to the value in the data file that is associated with the application. If the case exception file is not verbatim to the value, Exstream ignores the exception.

The ALLOW_CASE_EXCEPTION_PARTIAL_MATCH engine switch lets you circumvent this behavior in the software so that you can add case exceptions for phrases that do not contain spaces between words or numbers.

Syntax

-ALLOW_CASE_EXCEPTION_PARTIAL_MATCH

This switch does not have any arguments.

Supported in

7.0.601 and later versions
