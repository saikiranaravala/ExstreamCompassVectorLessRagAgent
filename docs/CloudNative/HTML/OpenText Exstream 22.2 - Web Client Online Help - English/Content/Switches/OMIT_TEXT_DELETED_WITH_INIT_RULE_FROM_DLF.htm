OMIT_TEXT_DELETED_WITH_INIT_RULE_FROM_DLF

Important: Starting with the Exstream 20.2 release, support for Exstream Live has been discontinued within the containerized Exstream environment. Live switches that are listed in the documentation are for reference only and are no longer supported.

To ensure that the engine deletes excluded text in DLF output, use the OMIT_TEXT_DELETED_WITH_INIT_RULE_FROM_DLF engine switch when you produce DLF output from an application that includes rules to control the inclusion of text.

Syntax

-OMIT_TEXT_DELETED_WITH_INIT_RULE_FROM_DLF

This switch does not have any arguments.

Supported in

7.0.626 and later versions
